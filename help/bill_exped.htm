Экспедирование по исходящему счету создается по счету, у которого утверждена отгрузка.

Для создания экспедирования необходимо нажать кнопку в карте счета, либо в строке этого счета в реестре исходящих счетов:

Рис. Кнопка создания экспедирования в карте счета.

Рис. Кнопка создания экспедирования в реестре исходящих счетов.

Откроется окно создания экспедирования:

Рис. Окно создания экспедирования.

В окне необходимо заполнить поля:

  • счета (1) -отметьте те исходящие счета, по которым производится экспедирование. По умолчанию уже отмечен счет, из которого вызвано создание экспедирования
  • Выбрать время работы в блоке (2). В поле "Общее время работы" время рассчитывается автоматически.
  • выбрать код оплаты из справочника, нажав кнопку "..." в блоке (3)
  • кол-во принятых машин (4)
  • указать при необходимости наличие, размер и причину премии руководителя (5)
  • выбрать из справочника экспедитора, нажав кнопку "..." в блоке (6). В справочник попадают те сотрудники, вопросы, курируемые которыми, включают в себя слово "экспедитор".
  • выбрать сотрудника-получателя платежа (7).

В поле (8) будет автоматически рассчитана стоимость экспедирования.

Заполним все необходимые поля:

Рис. Создание экспедирования: заполнены поля документа.

 

После заполнения всех полей нажмите кнопку "Готово" (9). При корректном заполнении полей будет создан документ вида Расход наличных на экспедирование и автоматически утверждены затраты в этом расходе наличных. Подробнее о расходе наличных Вы можете прочесть в разделе Расход наличных.

Программа выдаст сообщение "Экспедирование по счету создано!". Карта счета автоматически обновится.

Связанное со счетом экспедирование можно увидеть в реестре расходов наличных, а также на вкладке "Затраты по сделке" в карте исходящего счета:

Рис. Вкладка "Затраты по сделке" в карте исходящего счета.

При попытке повторно создать по этому исходящему счету экспедирование программа выдаст подтверждение, в котором будет указана ранее созданное экспедирование:

Рис. Подтверждение создания экспедирования по исходящему счету, по которому уже была ранее создано экспедирование.

При клике по кнопке у экспедирования на вкладке "Затраты по сделке" в карте исходящего счета будет открыта карта соответствующего расхода наличных. Подробнее об общей логике карты Вы можете прочесть в разделе "Расход наличных". Отметим, что поля карта повторяют поля окна создания экспедирования:

Рис. Карта расхода наличных.
