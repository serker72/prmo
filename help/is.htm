При наличии у Вас соответствующих прав Вам доступен раздел «Межсклад». В этом разделе показан список распоряжений на межсклад.

Данный раздел предназначен для оформления перемещения товаров между участками, объектами.

Рис. 5.64 Фрагмент окна раздела «Межсклад».

Если в списке распоряжений на межсклад видна кнопка «Создать распоряжение на межсклад…» (1), то у Вас есть права на создание распоряжения на межсклад.

Распоряжения на межсклад можно искать в списке по полям (2):

  • Номер
  • Дата создания
  • Заданный номер
  • Участок-отправитель
  • Объект-отправитель
  • Участок-получатель
  • Объект-получатель
  • Статус
  • Кто создал

Каждое распоряжение на межсклад в программе имеет два утверждения (3):

  • Утверждение отгрузки
  • Утверждение списания.

Подробнее об этих утверждениях будет рассказано в следующей главе.

С помощью кнопки  у конкретного распоряжения на межсклад  можно просмотреть или отредактировать данное распоряжение на межсклад.

С помощью кнопки   можно аннулировать распоряжение на межсклад.

Если кнопка серого цвета - , то аннулировать данное распоряжение на межсклад невозможно. Это может быть по нескольким причинам:

  • У Вас нет прав на аннулирование распоряжения на межсклад;
  • По распоряжению на межсклад есть связанные утвержденные документы
  • распоряжение на межсклад не в статусе «не утверждено».

При нажатии кнопки  будет выведено сообщение со списком причин, по которым невозможно аннулировать это распоряжение на межсклад.

Зеленым цветом выделены распоряжения на межсклад, находящиеся в статусе «выполнено», по которым проведены все документы: т.е. есть связанное поступление в статусе «выполнено», и утверждено наличие оригиналов всех документов поступления.
