Рис. 5.52 Окно просмотра/редактирования исходящей оплаты

Рассмотрим блоки окна редактирования оплаты.

Блок «статус» – на Рис. 5.52 имеет значение «не выполнена». У каждой исходящей оплаты в данном блоке указан ее статус.

Статус оплаты меняется следующим образом:

  • Оплату создали, но не утвердили – статус «не выполнена».
  • Оплату утвердили – статус «выполнена».
  • Аннулировали оплату (только из статуса «не выполнена») – статус «Аннулирована».

Блок «Вид оплаты» (2).

  • Оплата по договору. При выборе данного режима сумма оплаты распределяется автоматически по самым старым неоплаченным входящим счетам заданного поставщика. Платежи по найденным входящим счетам можно редактировать с помощью кнопки «Выбрать счета…».
  • Оплата по счету. При выборе данного режима сумма оплаты расходуется на оплату самого старого неоплаченного входящего счета поставщика. Выбрать другой входящий счет для оплаты можно с помощью кнопки «Выбрать счета…».

Не привязанный ко входящим счетам остаток оплаты будет зачислен как авансовый платеж по карте поставщика. При утверждении цен нового входящего счета данного поставщика будет предложено привязать авансовый платеж к счету.

Выберем режим «Оплата по договору». Сумма оплаты будет автоматически распределена по неоплаченным входящим счетам контрагента-поставщика:



Рис. 5.53 Окно просмотра/редактирования исходящей оплаты. Режим «оплата по договору».

Блок (3) – Примечания полностью аналогичен блоку «Примечания» в разделе Просмотр и редактирование заявки.

Утверждение оплаты (4) – при наличии у Вас прав Вы можете утвердить исходящую оплату.

Для этого необходимо проставить этот флаг и нажать кнопку «Сохранить и остаться» (4).
Для утверждения оплаты необходимо заполнить поля «Заданная дата» и «Заданный номер».
Данное действие так же возможно провести при наличии прав из реестра оплат. Для этого служит кнопка «утвердить» в колонке «Утверждение».

При утверждении исходящая оплата  переходит в статус «Выполнена».
Редактирование оплаты в статусе «Выполнена» невозможно!
Для редактирования исходящей оплаты необходимо снять утверждение!
