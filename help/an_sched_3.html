Отчет позволяет получить данные о встречах, доступных Вам в планировщике.

Рис. Форма отчета Встречи.

Доступны следующие фильтры и кнопки:

  1. Период - встречи отбираются по попаданию даты встречи в указанный период
  2. Отбор по статусам встреч
  3. Отбор по видам (месту) встреч
  4. Отбор запланированных и/или фактических встреч
  5. Отбор по сотруднику
  6. Отбор по контрагенту встречи. Галочка "Включая субхолдинги и подчиненные предприятия" позволяет выбрать в отчет также данные по субхолдингам и подчиненным предприятиям выбранных контрагентов.
  7. Отбор по виду контрагента
  8. Отбор по стране встречи
  9. Отбор по федеральному округу встречи. Функция доступна только для России. При выборе федерального округа автоматически выбирается страна Россия.
  10. Отбор по городу встречи
  11. Отбор по фрагменту описания результата встречи.
  12. Отбор по месту встречи
  13. Искать по фрагменту в приложенных файлах. Поиск по вложенным файлам доступен для файлов следующих типов: txt, docx, xlsx, xls, csv, pptx.
  14. Показать отчет по выбранным фильтрам
  15. Сбросить все фильтры
  16. Получить печатную версию отчета по выбранным фильтрам в pdf-формате.
