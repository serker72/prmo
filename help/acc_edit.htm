Рис. 5.42 Окно редактирования/просмотра реализации

Блоки (1)-(4) были рассмотрены ранее.

Блок (4) – Примечания полностью аналогичен блоку «Примечания» в разделе Просмотр и редактирование заявки.

Утверждение реализации (5) – при наличии у Вас прав Вы можете утвердить реализацию.

Для этого необходимо проставить этот флаг и нажать кнопку «Сохранить и остаться» (4).

Для утверждения реализации необходимо заполнить поля «Заданный номер с/ф» и «Заданная дата с/ф».

Данное действие так же возможно провести при наличии прав из реестра реализаций. Для этого служит кнопка «утвердить» в колонке «Статус».

Реализация  переходит в статус «Выполнено».

Редактирование реализации в статусе «Выполнено» невозможно!

Для редактирования реализации необходимо снять утверждение!

Реализацию из нашего примера утвердили. Окно просмотра реализации выглядит следующим образом:

Рис. 5.43 Окно просмотра/редактирования реализации. Реализация утверждена.

Кнопка «Файлы…» (10) открывает файловый реестр, содержащий файлы, прикрепленные к поступлению. Данный раздел полностью аналогичен разделу «Файлы» в заявке. (см. Просмотр и редактирование заявки)

При утверждении реализации становится активной кнопка  («Печать») (4). Для реализации доступны три печатные формы:

  • Товарная накладная (только для реализаций, содержащих товары, т.е. все позиции номенклатуры, кроме групп "Услуги", "Услуги по заявкам")
  • Счет-фактура
  • Акт (только для реализаций, содержащих позиции групп номенклатуры «Услуги», "Услуги по заявкам").

При нажатии кнопки  Вам будет предложено отметить формы для печати. После чего необходимо нажать кнопку «Печать»:

Рис. 5.44 Выбор форм для печати утвержденной реализации.

При утверждении реализации становится доступной кнопка "Отправить pdf-документы на электронную почту" (3). Эта кнопка позволяет отправить печатные формы реализации в pdf-формате на выбранные адреса электронной почты. При ее нажатии откроется окно выбора вида документов и адресатов. Адресатов можно выбрать из списка контактов сотрудников и контрагента-покупателя:

Рис. Окно выбора видов документов и адресата для отправки pdf-документов на электронную почту.

Необходимо отметить требуемые виды документов, адресатов и нажать кнопку Готово.

 

Вкладка «Оригиналы документов» (9) активна в утвержденной реализации и позволяет указать, прикреплены ли к данной реализации оригиналы:

  • Товарной накладной (только для реализаций, содержащих товары, т.е. все позиции номенклатуры, кроме групп "Услуги", "Услуги по заявкам")
  • Акта (только для реализаций, содержащих позиции групп номенклатуры «Услуги», "Услуги по заявкам").

Вкладка выглядит следующим образом:

Рис. 5.45 Вкладка «Оригиналы документов» в реализации.

Для указания наличия оригинала необходимо проставить на вкладке соответствующий флажок и нажать кнопку «Сохранить».
