При наличии у Вас прав Вам доступен Отчет «Факт оплаты» входящий (кнопка  в меню программы, вкладка "Вход", "Факт оплаты".).

В отчете показаны частично оплаченные и полностью оплаченные входящие счета по умолчанию за прошлые 3 месяца.

Счет считается частично оплаченным, если:
а) по счету нет поступлений, но есть оплаты, при этом сумма оплат меньше суммы счета;
б) по счету есть поступления и есть оплаты, при этом сумма оплат меньше суммы по поступлениям счета;


Счет считается полностью оплаченным, если:
а) по счету нет поступлений, но есть оплаты, при этом сумма оплат равна сумме счета;
б) по счету есть поступления и есть оплаты, при этом сумма оплат равна сумме по поступлениям счета;

Рассмотрим форму отчета:

Рис. 6.34 Отчет «Факт оплаты»: форма отчета.

(1) – фильтр «Контрагент»: Вы можете осуществлять отбор входящих счетов по нескольким контрагентам.

Для этого в форму поиска нужно вписать фрагмент названия искомого контрагента. Программа выдаст список вариантов названий. Выберите в списке требуемого контрагента щелчком мыши по его названию. При необходимости отобрать в отчет счета по еще одному контрагенту - снова наберите фрагмент его названия и выберите контрагента из всплывшего списка вариантов названий контрагентов.

(2) – период: позволяет отбирать для отчета входящие счета с фактическими датами оплаты , попадающими в выбранный период.

(3) Фильтр "Только выполненные счета" позволяет показать только те входящие счета, позиции по которым полностью завезены (т.е. счета в статусе "выполнен").
(4) Фильтр "Только невыполненные счета" позволяет показать только входящие счета, позиции которых не завезены, либо завезны частично (т.е. счета в статусе "не выполнен", "утвержден", "оплачен", "не оплачен").
Одновременно может быть отмечен только один из этих двух фильтров.

(5) Фильтр "Только оплаченные счета" позволяет показать только полностью оплаченные счета.
(6) Фильтр "Только частично оплаченные счета" позволяет показать только те входящие счета, по которым:
а) по счету нет поступлений, но есть оплаты, при этом сумма оплат меньше суммы счета;
б) по счету есть поступления и есть оплаты, при этом сумма оплат меньше суммы по поступлениям счета.
Одновременно может быть отмечен только один из этих двух фильтров.

Фильтры можно сочетать, например:
Сочетание фильтров Только выполненные счета и Только оплаченные счета выдаст входящие счета в статусе "выполнен", которые были полностью оплачены.
Сочетание фильтров Только невыполненные счета и Только оплаченные счета выдаст входящие счета в статусах "не выполнен", либо "утвержден", которые были полностью оплачены.

(7) – кнопка  («найти») - осуществляет формирование отчета по указанным параметрам в фильтрах (1)-(5) .

(8) – кнопка  («сбросить все фильтры») – сбрасывает все параметры фильтров (1)-(5), формирует отчет согласно общей логике.

(9) – кнопка  («печать») – производится печать отчета согласно параметрам фильтров (1)-(5).

Пример.

Найдем все оплаченные счета поставщика «Англетер» за период с 01.12.2014 по 27.12.2014.

Для этого введем в поле «Контрагент» текст:

Англетер

Укажем соответствующий период и нажмем кнопку :

Рис. 6.35 Отчет «Факт оплаты»: результат поиска по названию поставщика.

Полученные результаты (Рис. 6.35) можно сортировать по:

  • № счета в программе
  • Сумме счета
  • Статусу счета
  • Контрагенту
  • Дате оплаты по договору
  • Фактической дате оплаты
