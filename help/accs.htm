При наличии у Вас соответствующих прав Вам доступен раздел «Поступления/реализации». В этом разделе на вкладке Реализации показаны реализации, с которыми Вы работаете в программе «Мобильный офис».

Рис. 5.37 Фрагмент окна «Реализации».

Фильтр статусов (1) позволяет отфильтровать список реализаций по статусам:

  • Режим «Выполненные реализации» - только реализации в статусе «выполнено»
  • Режим «Невыполненные реализации» - только реализации в статусе «не выполнено».
  • Режим «Аннулированные реализации» - только реализации в статусе «Аннулировано»
  • Режим «Все реализации» - реализации со всеми статусами.

Можно выделить несколько статусов, например: реализации только в статусах выполнено, не выполнено.

Нажатие кнопки  позволяет запомнить в программе выбранные вами статусы, и когда вы снова зайдете в раздел «Реализации», то будут показаны документы по сохраненному ранее набору статусов.

Для фильтрации реализаций по статусу необходимо отметить требуемые статусы и нажать кнопку «Найти» (2).

Кроме того, в списке можно искать реализации по полям:

  • Номер реализации
  • Дата создания
  • Заданная дата с/ф
  • Заданный номер с/ф
  • Покупатель
  • Склад
  • Кто создал.

В колонке «Статус» указан статус реализации.

В колонке «Утверждение» (3) при наличии прав доступна кнопка изменения утверждения/снятия утверждения.

С помощью кнопки  у конкретной реализации можно просмотреть или отредактировать данную реализацию.

С помощью кнопки   можно аннулировать реализацию.

Если кнопка серого цвета - , то аннулировать данную реализацию невозможно. Это может быть по нескольким причинам:

  • У Вас нет прав на аннулирование реализации;
  • Реализация не в статусе «не утверждено».

При нажатии кнопки  будет выведено сообщение со списком причин, по которым невозможно аннулировать эту реализацию.

 

 

Настройка колонок реестра

Все основные реестры программы могут быть индивидуально сконфигурированы: вы можете указать, какие колонки и в какой последовательности вы желаете видеть. Всегда можно вернуться к стандартному виду реестра. Рассмотрим работу этой функции на примере реестра контрагентов.

Для настройки реестра нажмите кнопку :

Откроется окно настройки. Слева указаны доступные колонки (1), справа - колонки, которые показаны в реестре (2):

Для перемещения столбцов между блоками 1 и 2 выделите требуемые столбцы (можно выделить несколько столбцов, нажав и удерживая ctrl при их выделении), и нажмите кнопку > для перемещения столбцов из доступных в видимые, либо кнопку < для перемещения из видимых в доступные столбцы.

Доступна сортировка видимых столбцов (в блоке 2). Для того, чтобы поднять или опустить столбец или их группу, выделите требуемые столбцы и нажмите для подъема или для спуска столбца.

Для того, чтобы сохранить внесенные изменения, нажмите Готово. Реестр обновится, и будут показаны выбранные вами столбцы в выбранной вами последовательности.

Для того, чтобы восстановить стандартный вид реестра, нажмите кнопку "Сделать стандартными".
