При наличии у Вас соответствующих прав Вам доступен раздел «Контрагенты». В этом разделе показан список контрагентов, с которыми осуществляется работа в программе «Мобильный офис».

Рис. 4.41 Фрагмент окна раздела «Контрагенты».

Если в списке контрагентов видна кнопка «Создать карту контрагента» (1), то у Вас есть права на создание контрагента.

Фильтр «Активные контрагенты» (2) позволяет демонстрировать в списке только контрагентов, карты которых активны, и доступные для работы в программе. Возможен просмотр всех контрагентов, в том числе неактивных. Для этого нужно снять отметку фильтра и нажать кнопку «Найти» (3).

Кнопка  (7) позволяет распечатать реестр контрагентов.

Кроме того, в списке можно искать контрагентов и сортировать список  по полям:

  • Код контрагента в программе,
  • ОПФ,
  • Полное наименование,
  • ИНН,
  • КПП,
  • Юридический адрес,
  • Ответственный сотрудник.

Поиск возможен по фрагменту названия. Например, если вписать в поле Полное наименование фрагмент «ЭХМЗ», и нажать «Найти» (3), то будут найдены все контрагенты, содержащие в названии этот фрагмент:

Рис. 4.42 Результаты поиска по разделу «Контрагенты».

В графе "Ответственный сотрудник" (4) у каждого контрагента указан один, или несколько, или ни одного сотрудника, которые являются ответственными за карту контрагента. При наличии специальных прав сотруднику доступно редактирование всех карт контрагентов. В рамках общих прав - сотрудник может редактировать только те карты контрагента, в которых он выбран в блоке "Ответственный сотрудник".

Кнопка «Схема проезда» (5) у каждого контрагента позволяет открыть файловый реестр, в котором хранятся схемы проезда к данному контрагенту.

Кнопка «Акты сверки» (6) у каждого контрагента позволяет открыть файловый реестр, в котором хранятся акты сверки по данному контрагенту.

С помощью кнопки  у конкретного контрагента можно просмотреть или отредактировать карту контрагента. Если кнопка - серого цвета, то Вы не можете редактировать карту этого контрагента. Для внесения правок обратитесь к ответственному сотруднику, он указан в графе (4). Если ответственный сотрудник не указан, просим обратиться к администраторам программы.

 

С помощью кнопки   можно удалить контрагента.

Если кнопка серого цвета - , то удалить данного контрагента невозможно. Это может быть по нескольким причинам:

  • У Вас нет прав на удаление контрагентов;
  • По данному контрагенту в программе есть какие-либо утвержденные документы.

 

 

Настройка колонок реестра

Все основные реестры программы могут быть индивидуально сконфигурированы: вы можете указать, какие колонки и в какой последовательности вы желаете видеть. Всегда можно вернуться к стандартному виду реестра. Рассмотрим работу этой функции на примере реестра контрагентов.

Для настройки реестра нажмите кнопку :

Откроется окно настройки. Слева указаны доступные колонки (1), справа - колонки, которые показаны в реестре (2):

Для перемещения столбцов между блоками 1 и 2 выделите требуемые столбцы (можно выделить несколько столбцов, нажав и удерживая ctrl при их выделении), и нажмите кнопку > для перемещения столбцов из доступных в видимые, либо кнопку < для перемещения из видимых в доступные столбцы.

Доступна сортировка видимых столбцов (в блоке 2). Для того, чтобы поднять или опустить столбец или их группу, выделите требуемые столбцы и нажмите для подъема или для спуска столбца.

Для того, чтобы сохранить внесенные изменения, нажмите Готово. Реестр обновится, и будут показаны выбранные вами столбцы в выбранной вами последовательности.

Для того, чтобы восстановить стандартный вид реестра, нажмите кнопку "Сделать стандартными".
