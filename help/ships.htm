При наличии у Вас соответствующих прав Вам доступен раздел «Распоряжения на приемку». В этом разделе показан список распоряжений на приемку для участков, с которыми Вы работаете в программе «Мобильный офис».

Рис. 5.31 Фрагмент окна «Распоряжения на приемку».

Фильтр статусов (1) позволяет отфильтровать список распоряжений на приемку по статусам:

  • Режим «Утвержденные распоряжения» - только распоряжения в статусе «Утверждено»
  • Режим «Неутвержденные распоряжения» - только распоряжения в статусе «Не утверждено»
  • Режим «Невыполненные распоряжения» - только распоряжения в статусе «не выполнено»
  • Режим «Выполненные распоряжения» - только распоряжения в статусе «выполнено»
  • Режим «Аннулированные распоряжения» - только распоряжения в статусе «Аннулировано»
  • Режим «Все распоряжения» - распоряжения со всеми статусами.

Можно выделить несколько статусов, например: распоряжения только в статусах утверждено, не выполнено.

Нажатие кнопки  позволяет запомнить в программе выбранные вами статусы, и когда вы снова зайдете в раздел «Распоряжения на приемку», то будут показаны распоряжения по сохраненному ранее набору статусов.

Для фильтрации заявок по статусу необходимо отметить требуемые статусы и нажать кнопку «Найти» (2).

Кроме того, в списке можно искать распоряжения на приемку по полям:

  • Номер распоряжения
  • Дата создания
  • Контрагент (поставщик)
  • Участок
  • Объект
  • Кто создал.

В колонке «Статус»  указан статус распоряжения.

В колонке «Утверждение» (3) при наличии прав доступна кнопка изменения утверждения/снятия утверждения.

С помощью кнопки  у конкретного распоряжения на приемку можно просмотреть или отредактировать данное распоряжение на приемку.

С помощью кнопки   можно аннулировать распоряжение на приемку.

Если кнопка серого цвета - , то аннулировать данное распоряжения на приемку невозможно. Это может быть по нескольким причинам:

  • У Вас нет прав на аннулирование распоряжения на приемку;
  • По данному распоряжению на приемку в программе есть какие-либо утвержденные документы.
  • распоряжение на приемку не в статусе «не утверждено».

При нажатии кнопки  будет выведено сообщение со списком причин, по которым невозможно аннулировать это распоряжение на приемку.
