При наличии у Вас прав Вам доступен "Отчет Рентабельность сделок" (кнопка  в меню программы).

Отчет показывает рентабельность по каждой реализованной позиции исходящих счетов.

Рассмотрим форму отчета:

Рис. 6.25 Отчет «Рентабельность сделок»: форма отчета.

(1) – период: позволяет отобрать позиции по заданной дате реализации..

(2) – фильтр "Покупатель" позволяет отобрать позиции по контрагенту-покупателю. Вы можете осуществлять отбор по нескольким контрагентам. Для этого в форму поиска нужно вписать фрагмент названия искомого контрагента. Программа выдаст список вариантов названий. Выберите в списке требуемого контрагента щелчком мыши по его названию. При необходимости отобрать в отчет данные по еще одному контрагенту - снова наберите фрагмент его названия и выберите контрагента из всплывшего списка вариантов названий контрагентов.

(3) – кнопка  («найти») - осуществляет формирование отчета по указанным параметрам в фильтрах (1)-(2) .

(4) – кнопка  («сбросить все фильтры») – сбрасывает все параметры фильтров (1)-(2), формирует отчет согласно общей логике.

(5) – кнопка  («печать») – производится печать отчета согласно параметрам фильтров (1)-(2).

(6) – кнопка  («Excel-версия») – позволяет получить Excel-версию отчета согласно параметрам фильтров (1)-(2).

Пример.

Сформируем отчет за период c 20.12.2014 по 26.12.2014:

Рис. 6.26 Отчет "Рентабельность сделок": пример отчета.

Как видно из рисунка, в отчете выдаются:

  • сумма закупки по каждой позиции - сумма согласно поступлениям
  • сумма продажи по каждой позиции - сумма согласно реализациям
  • расходы: транспорт, экспедирование, сумма кеша, выданный +/-
  • прибыль=сумма продажи - сумма закупки - расходы

Также выдаются итоговые значения этих показателей для всего отчета.
