В разделе «Объекты» представлены объекты, работающие с  программой «Мобильный офис».

Рис. 4.33 Фрагмент окна раздела «Объекты».

Если в списке объектов видна кнопка «Добавить объект…» (1), то у Вас есть права на создание объектов.

Фильтр «Активные объекты» (2) позволяет демонстрировать в списке только объекты, карты которых активны. Эти объекты доступны для работы в программе. Возможен просмотр всех объектов, в том числе неактивных. Для этого нужно снять отметку фильтра и нажать кнопку «Найти» (3).

Кроме того, в списке можно искать объекты и сортировать список  по полям:

  • Код объекта
  • Название
  • Фактический адрес.

Поиск возможен по фрагменту названия. Например, если вписать в поле Название фрагмент «Центральный», и нажать «Найти» (3), то будут найдены все объекты, содержащие в названии этот фрагмент. На рисунке показаны результаты этого запроса:

Рис. 4.34 Результаты поиска по разделу «Объекты».

Кнопка «Схема проезда» у каждого объекта позволяет открыть файловый реестр, в котором хранятся схемы проезда к данному объекту и прочая документация.

С помощью кнопки  у конкретного объекта можно просмотреть или отредактировать карту.

С помощью кнопки   можно удалить объект при наличии соответствующих прав.

Если кнопка удаления неактивна (), то это обозначает, что либо у Вас нет соответствующих прав, либо по данному объекту есть утвержденные документы.

Кнопка  (4) позволяет распечатать реестр объектов.
