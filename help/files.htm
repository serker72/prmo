Разделы «Справочная информация» и «Файлы и документы» по принципам работы полностью аналогичны друг другу. Рассмотрим разделы на примере раздела «Справочная информация».

Рис. 5.53 Основной фрагмент окна «Файлы и документы».

Раздел представляет собой реестр доступных для скачивания файлов. В реестре указаны:

  • имена файлов,
  • размер в мегабайтах,
  • кто из сотрудников загрузил файл,
  • описание каждого файла,
  • даты загрузки.

Чтобы скачать файл, необходимо либо кликнуть по имени файла, либо нажать кнопку Получить.

В реестре файлы могут быть размещены в папках. Папки всегда показаны в реестре над файлами. У файла значок - , у пустой папки - , у папки, содержащей файлы и/или подпапки - .

Чтобы перейти в папку, кликните по ее названию.

Блок (8) указывает, в какой папке Вы находитесь.

В зависимости от Ваших прав, у Вас могут быть показаны блоки и кнопки:

(1) Загрузить файл. С помощью этого блока Вы можете загрузить файл в реестр. Необходимо описать файл в поле «Описание файла», и затем нажать кнопку «Выберите и загрузите файл…». Откроется окно выбора файла. После выбора файл будет загружен в раздел.

(2) Создать папку – при наличии у Вас прав Вы можете создать папку в файловом реестре. При нажатии кнопки (2) откроется окно создания папки:

Рис. 5.54 Окно создания папки.

Впишите название папки и при необходимости – ее описание. Нажмите кнопку «Создать папку». Папка будет создана, и реестр файлов обновится автоматически.

Внимание! При создании папки Вам автоматически будут присвоены полные права на эту папку. Другие сотрудники не видят эту папку в реестре. Для того, чтобы дать доступ другим сотрудникам, необходимо нажать кнопку Поделиться (5) у данной папки. Подробнее о кнопке Поделиться - см. далее.

(3) Переместить – при наличии у Вас прав Вы можете перемещать выбранные файлы и папки из текущей папки в другую выбранную папку.

Для этого выберите необходимые файлы (и/или папки), отметив их галочками в колонке (6). Затем нажмите кнопку Переместить (3). Откроется окно выбора целевой папки:

Рис. 5.55 Окно выбора целевой папки при перемещении файлов.

Необходимо выбрать целевую папку в окне и нажать кнопку Переместить.

Выбранные файлы (и/или папки) будут перемещены, реестр файлов автоматически обновится.

В зависимости от Ваших прав у каждой папки Вам может быть показана кнопка   – управление доступом к папке. С помощью этой функции можно разграничивать права на все операции над файлами в каждой конкретной папке: ограничивать доступ в папку, добавление, редактирование файлов, создание подпапок и т.п. Права назначаются по согласованию с администраторами программы.

У каждого файла и папки (в зависимости от Ваших прав) есть кнопки:

  - редактировать описание файла (для папки – название и описание );

 - удалить файл (папку).

Работа кнопки "Поделиться" (5).

У созданных Вами папок доступна кнопка "Поделиться":

С помощью этой кнопки Вы можете поделиться этой папкой с другими сотрудниками. При ее нажатии откроется окно редактирования доступа к папке:

Отметьте галочками те права тех сотрудников, которые Вы хотите предоставить на эту папку. Снимите галочки на те права, которые Вы хотите забрать у сотрудников. Для сохранения изменений нажмите кнопку "Сохранить и остаться" - вы останетесь в этом окне, либо "Сохранить и выйти" - окно "Поделиться папкой" закроется. Для отмены внесенных, но не сохраненных изменений нажмите "Закрыть".

Примечания. Если некоторые галочки неактивны, то Вы не имеете полномочий для назначения таких прав.
