В разделе «Номенклатура» представлена вся номенклатура, введенная в программу «Мобильный офис».

Рис. 5.12 Фрагмент окна раздела «Номенклатура».

Если у вас видна кнопка «Добавить позицию» (1), то у Вас есть права на добавление позиций номенклатуры.

Весь каталог номенклатуры разбит на товарные группы, включающие в себя подгруппы 1 и 2 уровня. Например, Металлопрокат-Песок. Позиции могут присутствовать как в товарной группе, так и в ее подгруппах.

Если у Вас видна кнопка «Добавить товарную группу…» (2), то у Вас есть права на управление товарными группами.

По разделу «Номенклатура» доступен поиск по следующим полям:

  • Код позиции
  • Наименование
  • Товарная группа (и подгруппы)
  • ГОСТ/ТУ
  • Единицы измерения
  • Длина
  • Ширина
  • Высота/толщина
  • Диаметр

Кроме того, при наличии у вас прав вам доступен фильтр по активности позиций (3).

Доступны следующие варианты:

  • Активные позиции
  • Неактивные позиции
  • Все позиции

По умолчанию выбран фильтр «Активные позиции».

Для поиска позиций введите значения или их фрагменты в соответствующие поля и нажмите кнопку «Найти» (4).

Рис. 5.13 Пример поиска по каталогу номенклатуры. Все товары в группе «Горюче-смазочные материалы».

Кнопка «Файлы…» позволяет перейти в файловый реестр каждой позиции номенклатуры.

С помощью кнопки  у конкретной позиции можно просмотреть или отредактировать карту данной позиции.

С помощью кнопки   можно удалить позицию.

Если кнопка серого цвета - , то удалить данную позицию невозможно. Это может быть по нескольким причинам:

  • У Вас нет прав на удаление позиций;
  • По данной позиции в программе есть какие-либо утвержденные документы.

 

 

Настройка колонок реестра

Все основные реестры программы могут быть индивидуально сконфигурированы: вы можете указать, какие колонки и в какой последовательности вы желаете видеть. Всегда можно вернуться к стандартному виду реестра. Рассмотрим работу этой функции на примере реестра контрагентов.

Для настройки реестра нажмите кнопку :

Откроется окно настройки. Слева указаны доступные колонки (1), справа - колонки, которые показаны в реестре (2):

Для перемещения столбцов между блоками 1 и 2 выделите требуемые столбцы (можно выделить несколько столбцов, нажав и удерживая ctrl при их выделении), и нажмите кнопку > для перемещения столбцов из доступных в видимые, либо кнопку < для перемещения из видимых в доступные столбцы.

Доступна сортировка видимых столбцов (в блоке 2). Для того, чтобы поднять или опустить столбец или их группу, выделите требуемые столбцы и нажмите для подъема или для спуска столбца.

Для того, чтобы сохранить внесенные изменения, нажмите Готово. Реестр обновится, и будут показаны выбранные вами столбцы в выбранной вами последовательности.

Для того, чтобы восстановить стандартный вид реестра, нажмите кнопку "Сделать стандартными".
