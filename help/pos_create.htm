Создание позиции при наличии у Вас прав на данное действие производится с помощью кнопки «Добавить позицию…» в разделе «Номенклатура» (Рис. 5.12).

Кроме того, Вы можете добавить позицию с помощью кнопки быстрого доступа «Новая позиция» в шапке окна программы.

При нажатии кнопки «Добавить позицию …» открывается окно создания позиции, полностью аналогичное окну редактирования:

Рис. 5.16 Окно создания позиции.

После заполнения полей нажмите кнопку «Создать позицию» для сохранения изменений и перехода в раздел «Номенклатура» или «Создать позицию и перейти к редактированию», чтобы сохранить изменения и остаться в карте позиции.

Кнопка «Закрыть форму» позволяет закрыть карту без сохранения изменений.
