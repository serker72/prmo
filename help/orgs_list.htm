При наличии у Вас соответствующих прав Вам доступен раздел «Мои организации». В этом разделе показан список организаций, работающих с программой «Мобильный офис».

Рис. 1 Фрагмент окна раздела «Мои организации».

Если в списке организаций видна кнопка «Создать карту организации», то у Вас есть права на создание организации.

Фильтр «Активные организации» (1) позволяет демонстрировать в списке только организации, карты которых активны, и доступные для работы в программе. Возможен просмотр всех организаций, в том числе неактивных. Для этого нужно снять отметку фильтра и нажать кнопку «Найти» (2).

Кнопка  («Отчетные периоды) (3) позволяет перейти в раздел «Отчетные периоды», который позволяет закрывать отчетные периоды, и открывать при необходимости внесения изменений ранее закрытые периоды работы в программе «Мобильный офис».

Кнопка ("Распределить входящие оплаты по реализациям") (3) доступна при наличии у вас специальных прав. Эта функция позволяет вызвать автоматическое перераспределение входящих оплат по реализациям в программе. Это действие также производится автоматически раз в сутки, и при утверждении либо снятии утверждения реализации.

Кроме того, в списке можно искать организацию и сортировать список  по полям:

  • Код организации в программе,
  • ОПФ,
  • Полное наименование,
  • ИНН,
  • КПП,
  • Юридический адрес.

Поиск возможен по фрагменту названия. Например, если вписать в поле Полное наименование фрагмент «Строительная», и нажать «Найти» (3), то будут найдены все организации, содержащие в названии этот фрагмент. В примере для Рис. 1 – в списке будет показана только организация «Торговый Дом Строительная Ярмарка»:

Рис. 2 Результаты поиска по разделу Мои организации.

С помощью кнопки  у конкретной организации можно просмотреть или отредактировать карту данной организации.

С помощью кнопки   можно удалить организацию.

Если кнопка серого цвета - , то удалить данную организацию невозможно. Это может быть по нескольким причинам:

  • У Вас нет прав на удаление организации;
  • По данной организации в программе есть какие-либо утвержденные документы.

 

Настройка колонок реестра

Все основные реестры программы могут быть индивидуально сконфигурированы: вы можете указать, какие колонки и в какой последовательности вы желаете видеть. Всегда можно вернуться к стандартному виду реестра. Рассмотрим работу этой функции на примере реестра контрагентов.

Для настройки реестра нажмите кнопку :

Откроется окно настройки. Слева указаны доступные колонки (1), справа - колонки, которые показаны в реестре (2):

Для перемещения столбцов между блоками 1 и 2 выделите требуемые столбцы (можно выделить несколько столбцов, нажав и удерживая ctrl при их выделении), и нажмите кнопку > для перемещения столбцов из доступных в видимые, либо кнопку < для перемещения из видимых в доступные столбцы.

Доступна сортировка видимых столбцов (в блоке 2). Для того, чтобы поднять или опустить столбец или их группу, выделите требуемые столбцы и нажмите для подъема или для спуска столбца.

Для того, чтобы сохранить внесенные изменения, нажмите Готово. Реестр обновится, и будут показаны выбранные вами столбцы в выбранной вами последовательности.

Для того, чтобы восстановить стандартный вид реестра, нажмите кнопку "Сделать стандартными".
