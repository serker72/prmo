При наличии у Вас соответствующих прав Вам доступен раздел «Счета». В этом разделе на вкладке "Входящие счета" показан список входящих счетов.

Рис. 5.14 Фрагмент окна раздела «Входящие счета».

Фильтр статусов (1) позволяет отфильтровать список счетов по статусам:

  • Режим «Утвержденные счета» - только счета в статусе «утвержден»
  • Режим «Неутвержденные счета» - только счета в статусе «не утвержден»
  • Режим «Невыполненные счета» - только счета в статусе «не выполнен»
  • Режим «Выполненные счета» - только счета в статусе «выполнен»
  • Режим «Аннулированные счета» - только счета в статусе «Аннулирован»
  • Режим «Все счета» - счета со всеми статусами.

Можно выделить несколько статусов, например: счета только в статусах утвержден,  выполнен.

Нажатие кнопки  позволяет запомнить в программе выбранные вами статусы, и когда вы снова зайдете в раздел «Входящие счета», то будут показаны счета по сохраненному ранее набору статусов.

Для фильтрации счетов по статусу необходимо отметить требуемые статусы и нажать кнопку «Найти» (3).

Кроме того, в списке можно искать счета и сортировать список  по полям:

  • Номер счета
  • Дата счета
  • Дата счета поставщика
  • № вход. счета поставщика
  • Поставщик
  • Склад
  • Статус
  • Кто создал.

Каждый счет в программе имеет два утверждения (3):

  • Утверждение цен
  • Утверждение приемки.

Подробнее об этих утверждениях будет рассказано в следующей главе.

С помощью кнопки  у конкретного счета можно просмотреть или отредактировать данный счет.

С помощью кнопки   можно аннулировать счет.

Если кнопка серого цвета - , то аннулировать данный счет невозможно. Это может быть по нескольким причинам:

  • У Вас нет прав на аннулирование счета;
  • По данному счету в программе есть какие-либо утвержденные документы.
  • Счет не в статусе «не утвержден».

В этом случае при нажатии кнопки  программа выдаст сообщение с перечислением причин.

 

 

Настройка колонок реестра

Все основные реестры программы могут быть индивидуально сконфигурированы: вы можете указать, какие колонки и в какой последовательности вы желаете видеть. Всегда можно вернуться к стандартному виду реестра. Рассмотрим работу этой функции на примере реестра контрагентов.

Для настройки реестра нажмите кнопку :

Откроется окно настройки. Слева указаны доступные колонки (1), справа - колонки, которые показаны в реестре (2):

Для перемещения столбцов между блоками 1 и 2 выделите требуемые столбцы (можно выделить несколько столбцов, нажав и удерживая ctrl при их выделении), и нажмите кнопку > для перемещения столбцов из доступных в видимые, либо кнопку < для перемещения из видимых в доступные столбцы.

Доступна сортировка видимых столбцов (в блоке 2). Для того, чтобы поднять или опустить столбец или их группу, выделите требуемые столбцы и нажмите для подъема или для спуска столбца.

Для того, чтобы сохранить внесенные изменения, нажмите Готово. Реестр обновится, и будут показаны выбранные вами столбцы в выбранной вами последовательности.

Для того, чтобы восстановить стандартный вид реестра, нажмите кнопку "Сделать стандартными".
