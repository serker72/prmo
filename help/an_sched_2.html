Отчет позволяет получить данные о командировках, доступных Вам в планировщике.

Рис. Форма отчета Командировки.

Доступны следующие фильтры и кнопки:

  1. Период - командировки отбираются по попаданию даты начала командировки в указанный период
  2. Отбор по статусам командировок
  3. Отбор запланированных и/или фактических командировок
  4. Отбор по сотруднику
  5. Отбор по контрагенту командировок. Галочка "Включая субхолдинги и подчиненные предприятия" позволяет выбрать в отчет также данные по субхолдингам и подчиненным предприятиям выбранных контрагентов.
  6. Отбор по виду контрагентов
  7. Отбор по стране командировок
  8. Отбор по федеральному округу командировок. Функция доступна только для России. При выборе федерального округа автоматически выбирается страна Россия.
  9. Отбор по городу командировок
  10. Отбор по фрагменту описания результата командировки.
  11. Искать по фрагменту в приложенных файлах. Поиск по вложенным файлам доступен для файлов следующих типов: txt, docx, xlsx, xls, csv, pptx.
  12. Показать отчет по выбранным фильтрам
  13. Сбросить все фильтры
  14. Получить печатную версию отчета по выбранным фильтрам в pdf-формате.
