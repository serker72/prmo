При наличии у Вас соответствующих прав Вам доступен раздел «Заявки». В этом разделе показан список заявок для участков, с которыми Вы работаете в программе «Мобильный офис».

Рис. 5.1 Фрагмент окна раздела «Заявки».

Если в списке заявок видна кнопка «Создать заявку…» (1), то у Вас есть права на создание заявки.

Фильтр статусов (2) позволяет отфильтровать список заявок по статусам:

  • Режим «Утвержденные заявки» - только заявки в статусе «утверждена»
  • Режим «Неутвержденные заявки» - только заявки в статусе «не утверждена».
  • Режим «Новые заявки» - только заявки в статусе «новая».
  • Режим «Не выполненные заявки» - только заявки в статусе «не выполнена»
  • Режим «Выполненные заявки» - только заявки в статусе «выполнена»
  • Режим «Аннулированные заявки» - только заявки в статусе «Аннулирована»
  • Режим «Все заявки» - заявки со всеми статусами.

Можно выделить несколько статусов, например: заявки только в статусах утверждена, не выполнена.

Нажатие кнопки  позволяет запомнить в программе выбранные вами статусы, и когда вы снова зайдете в раздел «Заявки», то будут показаны документы по сохраненному ранее набору статусов.

Для фильтрации заявок по статусу необходимо отметить требуемые статусы и нажать кнопку «Найти» (3).

Кроме того, в списке можно искать заявки и сортировать список  по полям:

  • Код заявки
  • Контрагент
  • Даты исполнения
  • Кто создал.

С помощью кнопки  у конкретной заявки можно просмотреть или отредактировать данную заявку.

С помощью кнопки   можно аннулировать заявку.

Если кнопка серого цвета - , то аннулировать данную заявку невозможно. Это может быть по нескольким причинам:

  • У Вас нет прав на аннулирование заявки;
  • По данной заявке в программе есть какие-либо утвержденные поступления/реализации.
  • Заявка не в статусе «не утверждена».

В этих случаях при нажатии кнопки  программа выдаст сообщение с перечислением причин, по которым невозможно аннулировать заявку.

Заявки в статусе «новая» мигают в реестре заявок в графе «Статус» значком .

 

 

Настройка колонок реестра

Все основные реестры программы могут быть индивидуально сконфигурированы: вы можете указать, какие колонки и в какой последовательности вы желаете видеть. Всегда можно вернуться к стандартному виду реестра. Рассмотрим работу этой функции на примере реестра контрагентов.

Для настройки реестра нажмите кнопку :

Откроется окно настройки. Слева указаны доступные колонки (1), справа - колонки, которые показаны в реестре (2):

Для перемещения столбцов между блоками 1 и 2 выделите требуемые столбцы (можно выделить несколько столбцов, нажав и удерживая ctrl при их выделении), и нажмите кнопку > для перемещения столбцов из доступных в видимые, либо кнопку < для перемещения из видимых в доступные столбцы.

Доступна сортировка видимых столбцов (в блоке 2). Для того, чтобы поднять или опустить столбец или их группу, выделите требуемые столбцы и нажмите для подъема или для спуска столбца.

Для того, чтобы сохранить внесенные изменения, нажмите Готово. Реестр обновится, и будут показаны выбранные вами столбцы в выбранной вами последовательности.

Для того, чтобы восстановить стандартный вид реестра, нажмите кнопку "Сделать стандартными".
