Просмотреть карту объекта Вы можете, нажав на кнопку  у нужного объекта.

Откроется карта объекта:

Рис. 4.35 Карта объекта.

Рассмотрим отдельные поля и блоки карты объекта.

Блок «Примечания» (1) позволяет добавлять примечания по объекту.  Количество примечаний не ограничено. Для добавления примечаний нажмите кнопку «Добавить примечания…». Откроется окно создания/редактирования примечаний:

Рис. 4.36 Окно создания/редактирования примечаний.

Впишите текст примечания и нажмите кнопку «Добавить».

Примечания можно редактировать при помощи кнопки  справа от редактируемых примечаний.

Откроется окно, аналогичное Рис. 4.36.

При наличии прав возможно удаление примечаний.

Для удаления примечаний  нажмите кнопку  справа от этого примечания.

Флаг «Объект активен» (2) делает карту активной, т.е. данный объект можно будет использовать при создании документов в программе.

Снятие флага «Объект активен» (2)

Если объект был активен в программе, и Вы снимаете его активность, то при сохранении программа проверяет следующие условия:

1)      На этом объекте не должно быть товарных остатков. Если товарные остатки присутствуют, то сохранить снятие флага (2) невозможно, выводится окно с сообщением:

Рис. 4.37 Окно сообщения о невозможности снять активность объекта.

Для того, чтобы снять активность объекта, необходимо списать все товарные остатки на нем. Это можно сделать в разделе «Списание продукции».

Проверить наличие товарных остатков на объекте можно в отчете «Товары на объектах».

2)      Если товарных остатков нет, то программа снимет активность объекта. При этом:

-все документы по объекту с частичным завозом будут выровнены;

-все документы по объекту без завоза будут аннулированы.

Если такие документы есть в программе, то будет выведено подтверждение снятия активности объекта с перечислением выравниваемых и аннулируемых документов:

Рис. 4.38 Подтверждение снятия активности объекта с перечислением изменяемых документов.

При утвердительном ответе активность объекта будет снята; все перечисленные в подтверждении изменения будут внесены в программу.

Блок «Участки, работающие с объектом» (3) позволяет задать участки, работающие с данным объектом. Этот блок связан с блоком «Объекты, работающие с участком» в карте участка (Рис. 4.28).

Кнопка «Схема проезда…» (4) открывает файловый реестр, содержащий файлы схемы проезда на объект:

Рис. 4.39 Фрагмент окна реестра файлов «Схема проезда на объект».

Раздел представляет собой реестр доступных для скачивания файлов (2). В реестре указаны:

  • имена файлов,
  • размер в мегабайтах,
  • кто из сотрудников загрузил файл,
  • описание каждого файла,
  • даты загрузки.

Чтобы скачать файл, необходимо либо кликнуть по имени файла, либо нажать кнопку Получить.

В зависимости от Ваших прав, у Вас могут быть показаны блоки и кнопки:

(1) Загрузить файл. С помощью этого блока Вы можете загрузить файл в реестр. Необходимо описать файл в поле «Описание файла», и затем нажать кнопку «Выберите и загрузите файл…». Откроется окно выбора файла. После выбора файл будет загружен в раздел.

В блоке (2) – кнопки:

  - редактировать описание файла;

 - удалить файл.

После внесения изменений нажмите кнопку (5) «Сохранить и перейти к списку объектов», чтобы перейти в реестр объектов, или кнопку (6) «Сохранить и остаться», чтобы остаться в карте объекта.

Кнопка «Закрыть форму» (7) позволяет выйти из карты, не сохраняя изменения.
