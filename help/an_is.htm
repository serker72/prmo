При наличии у Вас соответствующих прав Вам доступен отчет «Межсклад» (кнопка  в меню программы).

Этот отчет показывает, какие товары в течение выбранного периода были переданы с выбранного участка и объекта-отправителя на выбранный участок , объект-получатель (или на все объекты участка-получателя).

Рассмотрим форму отчета:

 Рис. 6.52 Отчет «Межсклад»: форма отчета.

(1) – период: позволяет отбирать для отчета товары согласно поступлениям по межскладу, заданные даты которых попадают в выбранный период.

(2) – блок фильтров «Отправитель»: позволяет выбрать участок и объект-отправитель с помощью  фильтров «Участок-отправитель», «Объект-отправитель». Если отмечены фильтры «только активные участки» - то для выбора доступны только активные участки, аналогично – для фильтра только активные объекты.

Для формирования отчета необходимо выбрать участок и объект-отправитель.

(3) –  блок фильтров «Получатель»: позволяет выбрать участок и объект-получатель, аналогично блоку фильтров «Отправитель».

Для формирования отчета необходимо выбрать участок -получатель.

(4) – фильтр позиций: можно фильтровать товары по:

  • Коду позиции
  • Фрагментам наименования
  • Единицам измерения
  • Товарным группам
  • ГОСТ/ТУ
  • Длине
  • Ширине
  • Высоте/толщине
  • Диаметру

 Доступен выбор по нескольким значениям: для этого в форму поиска нужно вписать через точку с запятой фрагменты искомых названий или величин.

(5)  - фильтр «Расширенный отчет» позволяет получить отчет за период с разбивкой перемещенных товаров по заданным датам их поступлений.

(6) – кнопка  («найти») - осуществляет формирование отчета по указанным параметрам в фильтрах (1)-(5) .

(7) – кнопка  («сбросить все фильтры») – сбрасывает все параметры фильтров (1)-(5), формирует отчет согласно общей логике.

(8) – кнопка  («печать») – производится печать отчета согласно параметрам фильтров (1)-(5).

Доступны две печатные формы отчета:

  • Отчет «Межсклад»
  • Требование-накладная

При нажатии кнопки Вам будет предложено отметить формы для печати. После чего необходимо нажать кнопку «Печать»:

Рис. 6.53 Выбор форм для печати отчета.

Пример.

Сформируем отчет о перемещении товаров с участка Центральный склад, объекта Центральный склад на все объекты участка №1 в период с 1.06.2014 по 16.12.2014

И нажмем кнопку :

Рис. 6.54 Отчет «Межсклад»: результат поиска, сокращенная ведомость.

По каждому количеству товара в полученном отчете можно получить детализацию, по каким документам получено это количество .Для этого нужно кликнуть по требуемому количеству. 

Получим детализацию количества позиции 00157 Ведро оцинкованное (12 л) в колонке «Депо (площадка №24)»:

Рис. 6.55 Отчет «Межсклад»: детализация количества позиции отчета.

Для получения разбивки по датам необходимо отметить фильтр «Развернутый отчет» и нажать кнопку :

Рис. 6.56 Отчет «Межсклад»: развернутый отчет.
