Просмотреть карту контрагента Вы можете, нажав на кнопку  у нужного контрагента.

Откроется карта контрагента:

Рис. 4.46 Карта контрагента.

Флаг "Утверждаю заполнение" (1) делает невозможным редактирование карты. Все поля при установленном флаге становятся недоступными для редактирования.

Флаг «Карта активна» (2) делает карту активной, т.е. по данному контрагенту могут быть заведены документы в программе.

Логика работы галочек "Утверждаю заполнение" и "Карта активна"

  • если сняты обе галочки, то контрагент виден только в общем списке контрагентов. По нему невозможно создать документы и записи планировщика.
  • Галочка "Карта активна": простановка этой галочки позволяет заносить записи планировщика, связанные с этим контрагентом. Для простановки этой галочки необходимо заполнить следующие поля карты:
    1. Наименование
    2. Отрасль, подотрасль (если у отрасли есть подотрасли).
    3. Хотя бы одно контактное лицо с хотя бы одним телефоном и хотя бы одним адресом электронной почты
    4. Город
  • Галочка "Утверждаю заполнение": простановка этой галочки совместно с проставленной галочкой "карта активна" позволяет заводить по данному контрагенту документы в программе (счета, оплаты и т.п.). Для простановки этой галочки необходимо заполнить следующие поля:
    1. Юридический адрес
    2. Фактический адрес
    3. ИНН
    4. КПП
    5. Банковские реквизиты
    6. Должен быть указан руководитель, главный бухгалтер
    7. Должны быть заполнены поля для галочки "карта активна".

При наличии прав на редактирование у Вас будут доступны кнопки «Сохранить и перейти к списку контрагентов» (3) и «Сохранить и остаться» (4).

Кнопка «Закрыть форму» (5) закрывает форму без сохранения изменений.

На Рис. 4.46 поля карты невозможно редактировать, т.к. у карты утверждено заполнение. Чтобы внести изменения в карту, необходимо сначала снять флаг «Утверждаю заполнение» (1) и нажать кнопку «Сохранить и остаться» (4). Карта примет следующий вид:

Рис. 4.47 Карта контрагента в режиме редактирования.

Как видно на Рис. 4.47, поля стали доступными для редактирования.

Рассмотрим подробнее отдельные поля и блоки.

Кнопка «Учредительные документы», «Посмотреть договор», «Акты сверок», «Схема проезда», "Файлы" (1) открывают соответствующие файловые реестры контрагента.

Рассмотрим эти файловые реестре на примере реестра документов договора.

Рис. 4.48 Фрагмент окна реестра документов договора контрагента.

Раздел представляет собой реестр доступных для скачивания файлов (2). В реестре указаны:

  • имена файлов,
  • размер в мегабайтах,
  • кто из сотрудников загрузил файл,
  • описание каждого файла,
  • даты загрузки.

Чтобы скачать файл, необходимо либо кликнуть по имени файла, либо нажать кнопку Получить.

В зависимости от Ваших прав, у Вас могут быть показаны блоки и кнопки:

(1) Загрузить файл. С помощью этого блока Вы можете загрузить файл в реестр. Необходимо описать файл в поле «Описание файла», и затем нажать кнопку «Выберите и загрузите файл…». Откроется окно выбора файла. После выбора файл будет загружен в раздел.

(2) В блоке (2) – кнопки:

   - редактировать описание файла;

 - удалить файл.

Блок (2) - наличие оригиналов учредительных докуметов позволяет указать в карте контрагента наличие оригиналов учредительных документов.

Блок (3) - поставщик/покупатель - позволяет указать, является ли контрагент поставщиком, либо покупателем. В зависимости от выбора в этом блоке, доступны входящие, или исходящие договоры в блоке (4).

Важно! Блок (4) на Рис. 4.47: Договоры контрагента.

Блок представляет собой справочник договоров. Можно указать любое количество договоров.

  • если контрагент является поставщиком, то необходимо создать запись как минимум одного входящего договора и указать основной входящий договор. Основной входящий договор будет по умолчанию подставляться в документы при выборе контрагента в роли поставщика.
  • если контрагент является покупателем, то необходимо создать запись как минимум одного исходящего договора и указать основной исходящий договор. Основной исходящий договор будет по умолчанию подставляться в документы при выборе контрагента в роли покупателя.

Рис. 4.49 Блок «Договоры» в карте контрагента.

Новый договор создается с помощью кнопки справа над блоком договоров. При ее нажатии откроется окно создания договора:

Рис. 4.50 Окно создания/редактирования договора контрагента.

В этом окне нужно выбрать вид договора (входящий, исходящий), указать № договора, отсрочку по договору и вид дней отсрочки, указать дату договора. Галочка "Основной договор" позволяет указать, что этот договор является основным.
У контрагента может быть только один основной входящий договор, и один основной исходящий договор.
После ввода всех данных необходимо нажать кнопку Добавить.

Договор можно редактировать. Для этого служит кнопка  в блоке договора. Откроется окно, аналогичное Рис. 4.50.

При наличии прав возможно удаление договоров.

Для удаления используйте кнопку  справа в блоке договора.

При наличии оригинала договора необходимо отметить галочку «наличие оригинала договора» у соответствующего договора и нажать кнопку .

Блок (5) "Город контрагента" позволяет выбрать из справочника один или несколько городов контрагента.

Блок «Контактная информация» (6) представляет собой справочник контактов. Можно ввести любое количество контактных лиц с любым количеством контактов.

Ввести новое контактное лицо можно с помощью кнопки «Добавить контакты…»:

Рис. 4.52 Окно создания/редактирования контактного лица.

В этом окне нужно ввести Фамилию, имя, отчество, дату рождения контактного лица и его должность и нажать кнопку «Добавить».

Контактное лицо можно редактировать. Для этого служит кнопка  под данными контактного лица. Откроется окно, аналогичное Рис. 4.52.

Добавить контакт контактному лицу возможно при помощи кнопки  в колонке «Контакты» в строке данного контактного лица. Откроется окно создания/редактирования контакта:

Рис. 4.53 Окно создания/редактирования контакта.

Необходимо выбрать вид контакта и вписать его значение, затем нажать кнопку «Добавить». Введенные контакты можно редактировать с помощью кнопки  справа от соответствующего контакта. При этом откроется окно, аналогичное Рис. 4.53

При наличии прав возможно удаление контактов.

Для удаления контактного лица со всеми его контактами используйте кнопку  в крайней правой колонке справочника контактов.

Для удаления только одного контакта используйте кнопку  справа от этого контакта.

Блок (7) на Рис. 4.47:  Руководители компании-контрагента работает полностью аналогично этому блоку в карте организации. Смотрите раздел инструкции:  Просмотр карты организации.

Блок «Фактические адреса» (8) на Рис. 4.47.

В этом блоке вводятся фактические адреса контрагента. Этих адресов может быть несколько.

Для создания нового адреса нажмите кнопку «Добавить фактические адреса…». Откроется окно создания/редактирования фактического адреса:

Рис. 4.54 Окно создания/редактирования фактического адреса.

Необходимо вписать адрес и нажать кнопку «Добавить».

Адреса можно редактировать при помощи кнопки  справа от редактируемого адреса.

Откроется окно, аналогичное Рис. 4.54.

При наличии прав возможно удаление адресов.

Для удаления адреса  нажмите кнопку  справа от этого адреса.

Блок «Реквизиты» (9) на Рис. 4.47 позволяет задавать реквизиты контрагента.

Для создания новых реквизитов нажмите кнопку «Добавить реквизиты…».

Откроется окно создания/редактирования реквизитов:

Рис. 4.55 Окно создания/редактирования реквизитов.

Необходимо заполнить поля окна и нажать кнопку «Добавить».

Флаг «Основные реквизиты» позволяет назначить вводимые реквизиты основными для этой контрагента. Основные реквизиты подставляются по умолчанию в качестве реквизитов контрагента при создании различных документов в программе.

Реквизиты можно редактировать при помощи кнопки  справа от редактируемых реквизитов.

Откроется окно, аналогичное Рис. 4.55.

При наличии прав возможно удаление реквизитов.

Для удаления реквизитов  нажмите кнопку  справа от удаляемых реквизитов.

 

Блок «Ответственные сотрудники» (10) на Рис. 4.47 позволяет управлять списком ответственных за карту контрагента сотрудников. При наличии специальных прав сотруднику доступно редактирование всех карт контрагентов. В рамках общих прав - сотрудник может редактировать только те карты контрагента, в которых он выбран в блоке "Ответственный сотрудник".

При наличии у Вас специальных прав Вы можете редактировать список ответственных сотрудников. Для удаления сотрудника нажмите кнопку справа от его имени. Для добавления сотрудников нажмите кнопку "Добавить ответственного", отметьте галочками нужных сотрудников и нажмите кнопку "ОК". Сотрудники будут внесены в карту.

 

Блок «Отрасль», "Подотрасль" (11) на Рис. 4.47 позволяет выбрать отрасль и подотрасль контрагента из справочника программы. Нажмите кнопку "..." для выбора отрасли. Откроется окно справочника отраслей:

Рис. Справочник отраслей.

Чтобы выбрать отрасль, Вы можете:

  • начать писать ее название в поле (1), затем выбрать ее из всплывающей подсказки, и нажать кнопку "Готово" для подстановки отрасли в карту контрагента.
  • либо - выбрать отрасль, отметив ее в списке (2). У ряда отраслей введены подотрасли. После того, как Вы отметите требуемую отрасль в списке, нажмите кнопку "Готово" для подстановки отрасли в карту.

После выбора отрасли необходимо аналогичным образом выбрать подотрасль.

Блок «Основной холдинг», "Субхолдинг", "Не входит в холдинг" (12) на Рис. 4.47 позволяет выбрать холдинг, субхолдинг контрагента из других контрагентов в программе. Если отметить галочку "Не входит в холдинг", то выбор холдинга и субхолдинга снимается.

Вкладка "Действия контрагента"

В карте контрагента при наличии у Вас соответствующих прав Вам доступна вкладка "Действия контрагента":

В этой вкладке показаны все действия планировщика, связанные с этим контрагентом:

Для поиска и фильтрации по данным вкладки доступны следующие поля и кнопки:

  1. Период - записи отбираются по следующим датам:
    • Встречи - дата встречи
    • Задачи - крайний срок выполнения задачи
    • Звонки - дата звонка
    • Заметки - дата создания заметки
    • Командировки - дата начала командировки
  2. Отбор по связанному сотруднику (или группе сотрудников)
  3. Отбор по статусам записей
  4. Показать отчет по выбранным фильтрам
  5. Сбросить все фильтры
  6. Получить печатную версию отчета по выбранным фильтрам в pdf-формате.

По всем основным колонкам отчета доступна сортировка. Для сортировки по определенной колонке кликните по кнопке или в соответствующей колонке таблицы.
