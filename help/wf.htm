При наличии у Вас соответствующих прав Вам доступен раздел «Списание продукции». В этом разделе показан список распоряжений на списание.

Данный раздел предназначен для оформления списания товаров.

Рис. 5.81 Фрагмент окна раздела «Списание продукции».

Если в списке распоряжений на списание видна кнопка «Создать распоряжение на списание …» (1), то у Вас есть права на создание распоряжения на списание.

Распоряжения на списание можно искать в списке по полям (2):

  • Номер
  • Дата создания
  • Заданный номер
  • Участок списания
  • Объект списания
  • Статус
  • Кто создал

Каждое распоряжение на списание в программе имеет два утверждения (3):

  • Утверждение заполнения
  • Утверждение списания.

Подробнее об этих утверждениях будет рассказано в следующей главе.

С помощью кнопки  у конкретного распоряжения на списание можно просмотреть или отредактировать данное распоряжение на списание.

С помощью кнопки   можно аннулировать распоряжение на списание.

Если кнопка серого цвета - , то аннулировать данное распоряжение на списание невозможно. Это может быть по нескольким причинам:

  • У Вас нет прав на аннулирование распоряжения на списание;
  • распоряжение на списание не в статусе «не утверждено»;
  • этот акт создан программой автоматически при списании недостачи по межскладу

При нажатии кнопки  будет выведено сообщение со списком причин, по которым невозможно аннулировать это распоряжение на списание.
