Создание заявки при наличии у Вас прав на данное действие производится с помощью кнопки «Создать заявку…» в реестре заявок (Рис. 6.1).

Создание происходит в 2 этапа: создание заявки для контрагента-покупателя и окончательное заполнение и утверждение заявки.

При нажатии кнопки «Создать заявку…» открывается окно создания заявки:

Рис. 6.2 Окно создания заявки.

Рассмотрим поля и кнопки окна создания заявки.

Поле «Статус» (1) у вновь создаваемой заявки имеет значение «не утверждена». Щелкнув по кнопке , Вы можете отменить создание заявки.

В поле «Контрагент» (2) необходимо выбрать контрагента-покупателя по заявке.

В поле «Период» (3) необходимо указать период выполнения заявки. Заявка может быть создана только с сегодняшнего числа.

Флаги «Без автовыравнивания» и «Без автоаннулирования» (7) предназначен для исключения этой заявки из автоматического выравнивания и аннулирования заявок.

Заполним контрагента и период заявки:

Рис. 6.8 Заявка, готовая к созданию.

Нажмем кнопку «Создать заявку и перейти к утверждению». Откроется окно редактирования вновь созданной заявки.

Рис. 6.9 Созданная и сохраненная заявка.

Далее в заявку необходимо добавить позиции.

Для того, чтобы добавить в заявку позиции номенклатуры, нажмите кнопку «Добавить позиции…» (1).

Откроется окно выбора позиций заявки:

Рис. 6.3 Окно выбора позиций номенклатуры – сокращенная форма поиска.

В этом окне необходимо с помощью формы поиска (1) найти нужную номенклатуру.

При наличии у Вас соответствующих прав Вы можете создать новую позицию номенклатуры при помощи кнопки «Создать позицию». Это удобно, когда при создании заявки выясняется, что необходимой позиции нет в номенклатуре.

При нажатии этой кнопки в номенклатуру будет добавлена позиция с полями, как заполнено в форме поиска, и затем вновь созданная позиция автоматически будет отображена в блоке «Найденные позиции» (3) и доступна для включения в заявку.

Необходимо максимально полно заполнить данные о создаваемой позиции, поэтому при внесении позиции предпочтительнее воспользоваться развернутой формой поиска.

Доступны только активные позиции номенклатуры (т.е. те позиции, в карте которых отмечен флажок «Позиция активна»).

На Рис. 6.3 показана сокращенная форма поиска, в которой можно найти позиции номенклатуры только по фрагменту названия. С помощью кнопки  сокращенная форма разворачивается в развернутую форму:

Рис. 6.4 Окно выбора позиций номенклатуры – развернутая форма поиска.

С помощью кнопки  развернутая форма поиска сворачивается в краткую форму.

Пример.

Найдем товары группы «Мебель» и подгруппы "Пинскдрев". Для этого выберем эту товарную группу, подгруппу 1 уровня в развернутой форме поиска и нажмем «Найти» (2). В секции «Найденные позиции» (3) мы получим результаты:

Рис. 6.5 Результаты поиска номенклатуры по товарной группе.

Для того, чтобы выбрать в заявку определенные позиции, нужно заполнить им поле «Количество».

Кнопка «Добавить выбранное и продолжить поиск» (4) позволяет добавить в заявку выбранные позиции и остаться в окне выбора номенклатуры.

Кнопка «Добавить выбранное и закрыть» (5) позволяет добавить в заявку выбранные позиции и закрыть окно выбора номенклатуры.

Кнопка «Закрыть» закрывает окно выбора номенклатуры, в заявку ничего не заносится.

В качестве примера заполним количества у позиций 02726, 03802 и нажмем «Добавить выбранное и продолжить поиск».

Затем найдем все товары по запросу «диван». Для этого снимем выбор товарной группы, а в поле «наименование позиции» напишем «диван» и нажмем «Найти» (2). Получим следующее:

Рис. 6.6 Результаты поиска номенклатуры по наименованию.

Заполним количества у позиций 03811, 03812. Нажмем кнопку «Добавить выбранное и закрыть».

Получим заявку с выбранными нами позициями:

Рис. 6.7 Заявка с выбранными позициями.

Сохраним заявку с введенными позициями. Для этого нажмем "Сохранить и остаться".
