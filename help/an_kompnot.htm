При наличии у Вас прав Вам доступен отчет "Выполненные заявки" (кнопка  в меню программы, вкладка "Выполенные заявки".).

В этом отчете показаны заявки, позиции которых полностью завезены.

Рассмотрим форму отчета:

Рис. 6.3 Отчет «Выполненные заявки»: форма отчета.

(1) – период: заявки в отчет отбираются по заданным датам связанных поступлений, попадающим в выбранный период.

(2) – фильтр по наименованию позиции: можно фильтровать заявки по фрагменту названия позиции заявки. Кроме того, можно включать в поиск несколько названий, для этого в форму поиска нужно вписать через точку с запятой фрагменты искомых названий.

(3) – фильтр по контрагенту-покупателю: можно фильтровать заявки по названию контрагента-покупателя, указанном в заявке. Кроме того, можно включать в поиск несколько названий, для этого в форму поиска нужно вписать через точку с запятой фрагменты искомых названий.

(4) - кнопка  («расширенный поиск») - переключение к расширенной форме фильтра по позиции: заявки можно отбирать не только по названию входящих в них позиций, но и по другим параметрам (аналогично отбору позиций номенклатуры в заявке).

(5) – кнопка  («найти») - осуществляет поиск заявок по указанным параметрам в фильтрах (1)-(4) .

(6) – кнопка  («сбросить все фильтры») – сбрасывает все параметры фильтров (1)-(3), ищет все заявки согласно общей логике отчета.

(7) – кнопка  («печать») – производится печать отчета согласно параметрам фильтров (1)-(4).

(8) – кнопка  («Excel-версия») – позволяет получить Excel-версию отчета согласно параметрам фильтров (1)-(4).

Пример.

Выберем все выполненные заявки за период по умолчанию для покупателя ЭХМЗ. Для этого введем название покупателя в фильтре (3) и нажмем кнопку  (5):

Рис. 6.4 Отчет «Выполненные заявки»: результат поиска по покупателю.

Полученные результаты можно сортировать по:

  • Дате создания заявки
  • Покупателю
  • Текущему статусу заявки
