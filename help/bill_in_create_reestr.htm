Входящие счета только на услуги (Товарная группа "Услуги" в номенклатуре) можно создать напрямую из реестра входящих счетов при наличии у вас прав на это действие.

Для создания такого счета нажмите кнопку "Создать вход. счет..." в реестре входящих счетов:

Рис. Создание входящего счета из реестра.

Откроется окно создания входящего счета. Оно аналогично окну создания входящего счета из исходящего, за исключением:

  • при выборе позиций доступны только услуги без заявок (Товарная группа "Услуги" в номенклатуре)

Рис. Окно создания входящего счета из реестра счетов.

Выберем контрагента нажатием кнопки "..." в блоке 1. Откроется окно выбора контрагента.

Рис. Окно выбора контрагента.

По умолчанию показаны все контрагенты. С помощью полей "Код", "Полное наименование", "ИНН", "КПП", "Юридический адрес" вы можете найти требуемого контрагента. Для этого введите требуемые значения в поля поиска и нажмите "Найти" (1).

После нахождения требуемого контрагента в таблице выделите его в столбце "Код" (2) и нажмите "Готово" (3).

Найдем контрагента Англетер. Наберем "Англетер" в поле "Полное наименование", нажмем "Найти", выберем контрагента в столбце (2):

Рис. Выбор контрагента Англетер.

Нажмем Готово. Контрагент будет подставлен в карту счета. При этом автоматически выберутся основные реквизиты контрагента и основной входящий договор (при наличии этих документов):

Рис. Создание счета. Контрагент выбран.

Нажмем кнопку "Редактировать позиции" (2). Откроется окно выбора услуг для включения в счет:

Рис. Окно выбора услуг для включения в счет.

Выбор услуг производится аналогично выбору товаров, описанному в разделе "Создание входящего счета из исходящего счета". После выбора услуг необходимо нажать кнопку Готово. Услуги будут внесены в счет.

Выберем услугу 05231 2 шт, укажем цену, нажмем Готово.

Рис. Услуга включена в счет.

Заполним заданный № счета, укажем дату оплаты по договору. Нажмем "Создать входящий счет и перейти к утверждению".
Откроется окно просмотра/редактирования созданного счета.
