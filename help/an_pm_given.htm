При наличии у Вас соответствующих прав Вам доступен отчет «Отчет +/-: выдано» (кнопка  в меню программы).

В отчет попадают все утвержденные исходящие счета и их позиции, которые были полностью реализованы (количество позиции в счете совпадает с количеством позиций в реализациях), и по позициям которых +/- был выдан частично, либо полностью.

Если позиции в исходящем счете не были выровнены, то эти позиции не попадают в отчет.

Рассмотрим форму отчета:

Рис. 4.10 Отчет +/-:  выдано: форма отчета

(1-2) – период: в отчет отбираются исходящие счета по датам выдачи +/-, которые  попадают в выбранный период. По умолчанию период не ограничен. Дата выдачи +/- - это дата утверждения выдачи суммы по связанному расходу наличных.

(3) - фильтр «Покупатель»: Вы можете осуществлять отбор исходящих счетов по нескольким контрагентам-покупателям.

Для этого в форму поиска нужно вписать фрагмент названия искомого контрагента. Программа выдаст список вариантов названий. Выберите в списке требуемого контрагента щелчком мыши по его названию. При необходимости отобрать в отчет счета по еще одному контрагенту - снова наберите фрагмент его названия и выберите контрагента из всплывшего списка вариантов названий контрагентов.

(4) – группа фильтров по оплатам счетов:

Фильтр «оплаченные счета» - позволяет отбирать в отчет счета со 100% оплатой

Фильтр «неоплаченные счета» - позволяет отбирать полностью неоплаченные счета

Фильтр «частично оплаченные счета» - позволяет отобрать частично оплаченные счета.

(5) - фильтр «Зад. № с/ф»: позволяет отбирать для отчета исходящие счета по фрагменту заданного номера с/ф связанного поступления.

(6) - фильтр «№ исх. Сч. П-ка»: позволяет отбирать для отчета исходящие счета по фрагменту заданного номера счета.

 (7) – кнопка  («найти») - осуществляет формирование отчета по указанным параметрам в фильтрах (1)-(6) .

(8) – кнопка  («сбросить все фильтры») – сбрасывает все параметры фильтров (1)-(6), формирует отчет согласно общей логике.

(9) – кнопка  («печать») – производится печать отчета согласно параметрам фильтров (1)-(6).

Рис. 4.11 Отчет +/-: выдано

На Рис. 4.11 показан пример отчета.

Полученные результаты можно сортировать по:

  • № счета в программе
  • Дате счета
  • Покупателю
  • Статусу счета

В отличие от отчета «не выдано», отсутствует кнопка ввода выданного +/-.
