Внутренняя доставка по исходящему счету создается по счету, у которого утверждена отгрузка.

Для создания внутренней доставки необходимо нажать кнопку в карте счета, либо в строке этого счета в реестре исходящих счетов:

Рис. Кнопка создания внутренней доставки в карте счета.

Рис. Кнопка создания внутренней доставки в реестре исходящих счетов.

Откроется окно создания внутренней доставки:

Рис. Окно создания внутренней доставки.

В окне необходимо заполнить поля:

  • маршрут (1) - описание маршрута
  • счета (2) -отметьте те исходящие счета, по которым производится доставка. По умолчанию уже отмечен счет, из которого вызвано создание внутренней доставки
  • выбрать код оплаты из справочника, нажав кнопку "..." в блоке (3)
  • вес груза, кг (4)
  • кол-во точек загрузки-выгрузки (5)
  • указать при необходимости наличие и размер премии за дальность (6)
  • указать при необходимости наличие, размер и причину премии руководителя (7)
  • выбрать из справочника водителя, нажав кнопку "..." в блоке (8). В справочник попадают те сотрудники, вопросы, курируемые которыми, включают в себя слово "водитель".
  • выбрать сотрудника-получателя платежа (9).

В поле (10) будет автоматически рассчитана стоимость внутренней доставки.

Заполним все необходимые поля:

Рис. Создание внутренней доставки: заполнены поля документа.

 

После заполнения всех полей нажмите кнопку "Готово" (11). При корректном заполнении полей будет создан документ вида Расход наличных на внутреннюю доставку и автоматически утверждены затраты в этом расходе наличных. Подробнее о расходе наличных Вы можете прочесть в разделе Расход наличных.

Программа выдаст сообщение "Доставка по счету создана!". Карта счета автоматически обновится.

Связанную со счетом внутреннюю доставку можно увидеть в реестре расходов наличных, а также на вкладке "Затраты по сделке" в карте исходящего счета:

Рис. Вкладка "Затраты по сделке" в карте исходящего счета.

При попытке повторно создать по этому исходящему счету внутреннюю доставку программа выдаст подтверждение, в котором будет указана ранее созданная внутренняя доставка:

Рис. Подтверждение создания внутренней доставки по исходящему счету, по которому уже была ранее создана внутренняя доставка.

При клике по кнопке у доставки на вкладке "Затраты по сделке" в карте исходящего счета будет открыта карта соответствующего расхода наличных. Подробнее об общей логике карты Вы можете прочесть в разделе "Расход наличных". Отметим, что поля карта повторяют поля окна создания доставки:

Рис. Карта расхода наличных.
