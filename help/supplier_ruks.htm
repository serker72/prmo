Блок «Руководители организации» (1).

Руководитель (генеральный директор) и главный бухгалтер организации задаются при помощи реестра руководителей. Перейти в реестр можно нажатием кнопки «…».

Рис. 4.5 Реестр руководителей организации.

В реестре руководителей доступны следующие операции:

(1)   Создать новую запись руководителя

(2)   Вернуться в карту организации

(3)   Отредактировать имеющуюся запись

(4)   Удалить имеющуюся запись руководителя.

Поле «Дата начала действия» позволяет указать, с какой даты данный руководитель является действующим в программе. В карту организации в соответствующее поле подставляется активный руководитель по состоянию на сегодняшнюю дату. Такой руководитель выделен в реестре зеленым цветом.

В печатные формы документов автоматически подставляются руководители, активные по состоянию на дату этого документа.

Пример.

Добавим нового генерального директора с 1.05.2014. Для этого нажмем кнопку «Создать запись руководителя» (1) в реестре руководителей. Откроется форма создания руководителя:

Рис. 4.6 Форма создания руководителя.

Заполним поля в форме. Укажем дату 01.05.2014 в поле (1), укажем ФИО в поле (2), выберем должность в поле (3).

При необходимости можно прикрепить образец подписи кнопкой (4).

Нажмем «Сохранить и выйти». Реестр руководителей примет вид:

Рис. 4.7 Реестр руководителей организации: создан новый руководитель.

Как видно из реестра, руководитель Иванов А.В., действовавший в программе с 01.07.2012, стал неактуальным (1) (не выделен), а руководитель Кузнецов В.М., действующий с 01.05.2014, стал актуальным (2) (выделен зеленым цветом).

Тем не менее, во всех печатных формах документов до 01.05.2014 в качестве генерального директора будет выступать Иванов А.В.

Во всех печатных формах документов с 01.05.2014 в качестве генерального директора будет выступать Кузнецов В.М.

В аналогичном режиме работают записи по главному бухгалтеру.
