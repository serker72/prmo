Кнопка «Файлы…» (10) открывает файловый реестр, содержащий файлы, прикрепленные к поступлению. Данный раздел полностью аналогичен разделу «Файлы» в заявке. (см. Просмотр и редактирование заявки)

При утверждении поступления становится активной кнопка  («Печать») (3). Для поступления доступны три печатные формы:

  • Товарная накладная (только для поступлений, содержащих товары, т.е. все позиции номенклатуры, кроме групп "Услуги", "Услуги по заявкам")
  • Счет-фактура (если в карте контрагента-поставщика не проставлена галочка "Упрощенная форма налогообложения")
  • Акт (только для поступлений, содержащих позиции групп номенклатуры «Услуги», "Услуги по заявкам").

При нажатии кнопки  Вам будет предложено отметить формы для печати. После чего необходимо нажать кнопку «Печать»:

Рис. 5.44 Выбор форм для печати утвержденного поступления.

Вкладка «Оригиналы документов» (9) активна в утвержденном поступлении и позволяет указать, прикреплены ли к данному поступлению оригиналы:

  • Товарной накладной (активна для поступлений, содержащих товары, т.е. все позиции номенклатуры, кроме групп "Услуги", "Услуги по заявкам")
  • Счета-фактуры (если в карте контрагента-поставщика не проставлена галочка "Упрощенная форма налогообложения")
  • Акта (активна только в случае наличия в поступлении позиций из групп номенклатуры «Услуги», "Услуги по заявкам").

Снятие утверждения поступления при наличии связанных утвержденных реализаций

Важное замечание!

Если с поступлением связаны утвержденные реализации, то при снятии утверждения этого поступления:

  • программа выведет подтверждение снятия утверждения поступления с перечнем связанных реализаций
  • при положительном ответе на подтверждение будет снято утверждение поступления, и автоматически будет снято утверждение связанных реализаций. Эти документы перейдут в статус "не утверждено".
  • родительские входящий и исходящие счета перейдут в статус "не выполнено".
  • родительская заявка также перейдет в статус "не выполнена".

Снятие утверждения поступления при наличии связанных утвержденных реализаций, по позициям которых есть +/- и утверждена его выдача

Важное замечание!

Если с поступлением связаны утвержденные реализации, по позициям которых есть +/- и утверждена его выдача, то при снятии утверждения этого поступления:

  • будет выдано сообщение о невозможности снять утверждение этого поступления.
  • в этом сообщении будет содержаться перечень связанных расходов наличных по выдаче +/-.

Чтобы снять утверждение такого поступления, необходимо снять утверждение выдачи указанных расходов наличных.
