В этом диалоге Вы можете выбрать поставщика по входящему счету.

Чтобы выбрать поставщика для входящего счета, выполните двойной клик по строчке с требуемым поставщиком.
При этом поставщик будет подставлен в счет, автоматически подставятся № и дата договора с поставщиком, и будут выбраны основные реквизиты поставщика (если они были заданы в карте поставщика).
Для задания других (не основных) рекизитов поставщика воспользуйтесь кнопкой "..." возле поля "Банковский счет".

По умолчанию раскрыт полный список поставщиков. В этом списке можно найти необходимого Вам поставщика с помощью фильтров. Фильтры доступны по полям:
-Код поставщика
-Полное наименование
-ИНН
-КПП
-Юридический адрес.

Введите в одно или несколько из этих полей фрагмент данных нужного Вам поставщика и нажмите кнопку "Найти".
