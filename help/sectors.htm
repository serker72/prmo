В разделе «Склады» представлены склады, работающие в программе «Мобильный офис».

Рис. 4.26 Фрагмент окна раздела «Склады».

Если в списке участков видна кнопка «Добавить склад» (1), то у Вас есть права на создание склада.

Фильтр «Активные склады» (2) позволяет демонстрировать в списке только склады, карты которых активны. Эти склады доступны для работы в программе. Возможен просмотр всех складов, в том числе неактивных. Для этого нужно снять отметку фильтра и нажать кнопку «Найти» (3).

Кроме того, в списке можно искать участки и сортировать список  по полям:

  • Код склада
  • Название
  • Фактический адрес.

Поиск возможен по фрагменту названия. Например, если вписать в поле Название фрагмент «Белогород», и нажать «Найти» (3), то будут найдены все склады, содержащие в названии этот фрагмент. На рисунке показаны результаты этого запроса:

Рис. 4.27 Результаты поиска по разделу «Склады».

С помощью кнопки  у конкретного склада можно просмотреть или отредактировать карту.

С помощью кнопки   можно удалить склад при наличии соответствующих прав.

Если кнопка удаления неактивна (), то это обозначает, что либо у Вас нет соответствующих прав, либо по данному складу есть утвержденные документы.

Кнопка  (4) позволяет распечатать реестр складов.

 

 

Настройка колонок реестра

Все основные реестры программы могут быть индивидуально сконфигурированы: вы можете указать, какие колонки и в какой последовательности вы желаете видеть. Всегда можно вернуться к стандартному виду реестра. Рассмотрим работу этой функции на примере реестра контрагентов.

Для настройки реестра нажмите кнопку :

Откроется окно настройки. Слева указаны доступные колонки (1), справа - колонки, которые показаны в реестре (2):

Для перемещения столбцов между блоками 1 и 2 выделите требуемые столбцы (можно выделить несколько столбцов, нажав и удерживая ctrl при их выделении), и нажмите кнопку > для перемещения столбцов из доступных в видимые, либо кнопку < для перемещения из видимых в доступные столбцы.

Доступна сортировка видимых столбцов (в блоке 2). Для того, чтобы поднять или опустить столбец или их группу, выделите требуемые столбцы и нажмите для подъема или для спуска столбца.

Для того, чтобы сохранить внесенные изменения, нажмите Готово. Реестр обновится, и будут показаны выбранные вами столбцы в выбранной вами последовательности.

Для того, чтобы восстановить стандартный вид реестра, нажмите кнопку "Сделать стандартными".
