При наличии у Вас соответствующих прав Вам доступен отчет «Ведомость по контрагенту» (кнопка  в меню программы).

Этот отчет показывает баланс между суммами поступлений, реализаций, входящих, исходящих оплат по одному, нескольким, или всем контрагентам за выбранный период.

Рассмотрим форму отчета:

 Рис. 6.25 Отчет «Ведомость по контрагенту»: форма отчета.

(1) – фильтр «Контрагент»: Вы можете осуществлять отбор по нескольким контрагентам.

Для этого в форму поиска нужно вписать фрагмент названия искомого контрагента. Программа выдаст список вариантов названий. Выберите в списке требуемого контрагента щелчком мыши по его названию. При необходимости отобрать в отчет данные по еще одному контрагенту - снова наберите фрагмент его названия и выберите контрагента из всплывшего списка вариантов названий контрагентов.

·         Если Вы не выбрали ни одного контрагента в фильтре, то контрагенты, у которых за выбранный период начальный остаток, сумма поступлений, реализаций, сумма оплат (и конечный остаток) равны нулю, автоматически исключаются из отчета

·         Если Вы выбрали определенного контрагента, и у него за выбранный период начальный остаток, сумма поступлений, реализаций, сумма оплат (и конечный остаток) равны нулю, то контрагент включен в отчет с комментарием об отсутствии оборотов.

(2) – период: позволяет отбирать для отчета документы (реализации, поступления и оплаты) с заданными датами, попадающими в выбранный период.

(3) – фильтр «Расширенная  ведомость»: позволяет получить перечень документов движения по каждому выбранному контрагенту. Если этот фильтр не отмечен, то по каждому выбранному контрагенту показывается только одна строка с суммарными значениями; если фильтр отмечен, то показывается строка с суммарными значениями и перечислены все документы с сортировкой по заданным датам.

(4) - фильтр "с разбивкой по договорам": позволяет разбивать перечень документов движения по каждому выбранному контрагенту за каждый день на отдельные блоки по договорам с контрагеном.

(5) – фильтр «Все к-ты с похожим названием»: позволяет отбирать в ведомость контрагентов, в названиях которых содержатся введенные в фильтре «Контрагент» значения.

Например, если Вы ввели в фильтр название Текма и выбрали этот флажок, то в ведомость будут включены сведения по контрагенту Текма и по всем остальным контрагентам, в название которых входит слово Текма (такие, как Спектор (Текма) и т.д.).

(6) – кнопка  («найти») - осуществляет формирование отчета по указанным параметрам в фильтрах (1)-(5) .

(7) – кнопка  («сбросить все фильтры») – сбрасывает все параметры фильтров (1)-(5), формирует отчет согласно общей логике.

(8) – кнопка  («печать») – производится печать отчета согласно параметрам фильтров (1)-(5).

Доступны две печатные формы отчета:

  • ведомость по контрагенту
  • акт сверки взаиморасчетов

Печать акта сверки взаиморасчетов доступна, если выполнены все условия:

  • отчет сформирован по единственному контрагенту,
  • выбран режим Расширенная ведомость,
  • выбран определенный период.

При нажатии кнопки Вам будет предложено отметить формы для печати. После чего необходимо нажать кнопку «Печать»:

Рис. 6.26 Выбор форм для печати отчета.

Пример.

Сформируем ведомость по контрагенту «Инда Хаус» за период с 11.12.2014 по 26.12.2014. Для этого введем в поле «Контрагент» текст:

Инда Хаус,

укажем выбранный период

И нажмем кнопку :

Рис. 6.27 Отчет «Ведомость по контрагенту»: результат поиска по контрагенту, сокращенная ведомость.

Для получения списка документов движения необходимо отметить фильтр «Расширенная ведомость» и нажать кнопку :

Рис. 6.28 Отчет «Ведомость по контрагенту»: результат поиска по контрагенту, расширенная ведомость.
