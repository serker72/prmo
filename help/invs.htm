Данный модуль предназначен для инвентаризации остатков на участках и объектах.

Он доступен в меню «Инвентаризация» на вкладке «Инвентаризация складских остатков»:

Рис. 5.9.1 Фрагмент реестра актов инвентаризации остатков.

Если в списке оплат видна кнопка «Создать инвентаризационный акт…» (1), то у Вас есть права на создание акта.

Фильтр статусов (2) позволяет отфильтровать список актов по статусам:

  • Режим «Утвержденные акты» - только акты в статусе «утвержден»
  • Режим «Неутвержденные акты» - только акты в статусе «не утвержден»
  • Режим «Выполненные акты» - только акты в статусе «выполнен»
  • Режим «Аннулированные акты» - только акты в статусе «Аннулирован»
  • Режим «Все акты» - акты со всеми статусами.

Можно выделить несколько статусов, например: акты только в статусах утвержден, выполнен.

Нажатие кнопки  позволяет запомнить в программе выбранные вами статусы, и когда вы снова зайдете в раздел «Инвентаризация», то будут показаны документы по сохраненному ранее набору статусов.

Кроме того, в списке можно искать оплаты  по полям:

  • Номер
  • Дата инвентаризации
  • Заданный номер
  • Склад инвентаризации
  • Статус
  • Кто создал

С помощью кнопки  у конкретного акта можно просмотреть или отредактировать данный акт.

С помощью кнопки   можно аннулировать акт.

Если кнопка серого цвета - , то аннулировать данный акт невозможно. Это может быть по нескольким причинам:

  • У Вас нет прав на аннулирование акта;
  • акт не в статусе «не утвержден».

При нажатии кнопки  будет выведено сообщение со списком причин, по которым невозможно аннулировать этот акт.

 

 

Настройка колонок реестра

Все основные реестры программы могут быть индивидуально сконфигурированы: вы можете указать, какие колонки и в какой последовательности вы желаете видеть. Всегда можно вернуться к стандартному виду реестра. Рассмотрим работу этой функции на примере реестра контрагентов.

Для настройки реестра нажмите кнопку :

Откроется окно настройки. Слева указаны доступные колонки (1), справа - колонки, которые показаны в реестре (2):

Для перемещения столбцов между блоками 1 и 2 выделите требуемые столбцы (можно выделить несколько столбцов, нажав и удерживая ctrl при их выделении), и нажмите кнопку > для перемещения столбцов из доступных в видимые, либо кнопку < для перемещения из видимых в доступные столбцы.

Доступна сортировка видимых столбцов (в блоке 2). Для того, чтобы поднять или опустить столбец или их группу, выделите требуемые столбцы и нажмите для подъема или для спуска столбца.

Для того, чтобы сохранить внесенные изменения, нажмите Готово. Реестр обновится, и будут показаны выбранные вами столбцы в выбранной вами последовательности.

Для того, чтобы восстановить стандартный вид реестра, нажмите кнопку "Сделать стандартными".
