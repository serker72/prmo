При наличии у Вас прав Вам доступен Отчет "Оригиналы отгрузочных документов - исходящие" (кнопка  в меню программы, вкладка "Оригиналы отгрузочных документов - исходящие".).

В этом отчете показаны реализации, в картах которых не было отмечено наличие оригиналов документов.

Рассмотрим форму отчета:

Рис. 6.21 Отчет «Оригиналы документов»: форма отчета.

(1) – фильтр «нет оригиналов»: позволяет отбирать для отчета документы с отсутствующими оригиналами определенных документов. По умолчанию выбрано отсутствие всех оригиналов документов.

(4) - Фильтр "Контрагент".

Вы можете получить отчет по нескольким контрагентам.

Для этого в форму поиска нужно вписать фрагмент названия искомого контрагента. Программа выдаст список вариантов названий. Выберите в списке требуемого контрагента щелчком мыши по его названию. При необходимости отобрать в отчет документы по еще одному контрагенту - снова наберите фрагмент его названия и выберите контрагента из всплывшего списка вариантов названий контрагентов.

 (3) – кнопка  («найти») - осуществляет поиск поступлений по указанным параметрам в фильтрах (1)-(2) .

(4) – кнопка  («сбросить все фильтры») – сбрасывает все параметры фильтров (1)-(2), ищет все реализации согласно общей логике отчета.

(5) – кнопка  («печать») – производится печать отчета согласно параметрам фильтров (1)-(2).

Пример.

Найдем реализации без оригиналов всех документов для контрагента« Инда Хаус ». Для этого впишем:

Инда Хаус

в поле «Контрагент», затем нажмем кнопку :

Рис. 6.22 Отчет «Оригиналы документов»: результат поиска по контрагенту.

Полученные результаты (Рис. 6.22) можно сортировать по:

  • № документа
  • Дате создания
  • Заданной дате с/ф
  • Контрагенту
  • Статусу документа.
