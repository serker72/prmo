Создание распоряжения на приемку

Распоряжения на приемку в программе «Мобильный офис» создаются из входящих счетов (раздел «Входящие счета»).

Создание распоряжения на приемку при наличии у Вас прав на данное действие производится с помощью кнопки  («создать новое распоряжение на приемку …») в карте счета (Рис. 5.27).

Рассмотрим создание распоряжения на приемку на примере счета, созданного в предыдущей главе.

Для создания распоряжения на приемку необходимо отметить в счете те позиции, которые необходимо включить в распоряжение:

Рис. 5.29 Создание распоряжения на приемку из счета. Позиции, отмеченные для включения в распоряжение.

На Рис. 5.15 отмечены позиции счета  (1). В программе можно создать распоряжение на приемку, включающее в себя часть позиций счета. Оставшиеся позиции можно включить в другие распоряжения на приемку. Таким образом, по одному счету может быть создано несколько распоряжений на приемку.

Внимание! Допускается создание распоряжения на приемку только для позиций счета с одинаковыми объектами и участками.

После того, как Вы отметили позиции счета, необходимо нажать кнопку   (2). В результате откроется окно создания распоряжения на приемку:

Рис. 5.30 Окно создания распоряжения на приемку.

Рассмотрим поля и блоки окна создания распоряжения на приемку.

В блоке «Статус» (1) указан статус распоряжения.

Статус меняется следующим образом:

  • Распоряжение создали, но не утвердили – статус «не утвержден».
  • Распоряжение утвердили – статус «утвержден».
  • По распоряжению есть утвержденные связанные поступления, но все позиции не завезены – статус «не выполнен».
  • Все позиции распоряжения завезены или завезено больше (в рамках 10% превышения) – статус «выполнен».
  • Аннулировали распоряжения (только из статуса «не утвержден») – статус «аннулирован».

С помощью кнопки  (3 на Рис. 5.30) возможно редактирование количества позиций распоряжения на приемку. Для этого необходимо выделить нужные позиции и нажать кнопку . Редактирование количества полностью аналогично этой функции в счете и заявке.

Блок «Позиции распоряжения на приемку» .

Данный блок носит информационный характер.

В колонке «Количество» указано количество позиций в данном распоряжении.

В колонке «Доступно» указано не завезенное количество позиций по данному распоряжению.

В колонке «Получено» указано завезенное количество позиций. Доступна детализация по каждой цифре данной колонки. Для получения детализации необходимо щелкнуть мышью по интересующему количеству.

С помощью кнопки  можно удалить позицию из распоряжения на приемку.

Кнопка «Редактировать позиции…» открывает окно с информационной таблицей по позициям распоряжения на приемку:

Рис. 5.31 Информационная таблица позиций распоряжения на приемку.

Вернемся к Рис. 5.30.

Кнопка  («создать новое поступление») (2) служит для создания поступления выделенных позиций.

Если кнопка имеет вид  - неактивна, то поступление создать невозможно по одной из причин:

  • Распоряжение на приемку не утверждено
  • У Вас нет прав на создание поступления.

Кнопка  (4) – «Печать распоряжения на приемку» - позволяет распечатать распоряжение.

Кнопка  (5) – «Выровнять позиции с фактически полученными» - позволяет провести выравнивание позиций счета в соответствии со связанными с распоряжением на приемку  поступлениями. Для этого необходимо отметить нужные позиции и нажать эту кнопку.

Если кнопка имеет вид  - неактивна, то выравнивание позиций произвести невозможно по одной из причин:

  • Распоряжение на приемку не утверждено
  • У Вас нет прав на выравнивание позиций.

Кнопка «Создать распоряжение» (6) позволяет создать распоряжение с введенными данными и перейти в реестр распоряжений.

Кнопка «Создать распоряжение и перейти к утверждению» (7) позволяет создать распоряжение с введенными данными и остаться в карте распоряжения с возможностью утвердить распоряжение (если у Вас есть права на данное действие).

Кнопка «Закрыть форму» (8) позволяет закрыть форму, не сохраняя изменения.

Вкладка «Поступления» (9) содержит реестры связанных с распоряжением на приемку поступлений.

Нажмем кнопку «Создать распоряжение и перейти к утверждению» (7).

Откроется окно просмотра/редактирования созданного распоряжения на приемку.
