Период










1. В отчет попадают все утвержденные исходящие счета и их позиции, которые были полностью реализованы (количество позиции в счете совпадает с количеством позиций в реализациях), и по позициям которых +/- не был выдан, либо был выдан частично.

2. Фильтр "Период". По умолчанию отчет формируется изо всех документов по п.1 за все время работы программы. При выборе периода и нажатии кнопки программа отбирает исходящие счета по фактическим заданным датам поступления по счету, попадающим в выбранный период.
3. Фильтр "Покупатель". Вы можете осуществлять отбор исходящих счетов по нескольким контрагентам-покупателям.
Для этого в форму поиска нужно вписать фрагмент названия искомого контрагента. Программа выдаст список вариантов названий. Выберите в списке требуемого контрагента щелчком мыши по его названию. При необходимости отобрать в отчет счета по еще одному контрагенту - снова наберите фрагмент его названия и выберите контрагента из всплывшего списка вариантов названий контрагентов.
После выбора всех необходимых контрагентов нажмите кнопку .
