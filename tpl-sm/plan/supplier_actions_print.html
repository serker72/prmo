В этом диалоге Вы можете выбрать контрагента по входящему счету.

Чтобы выбрать контрагента для входящего счета, выполните двойной клик по строчке с требуемым контрагентом.
При этом контрагент будет подставлен в счет, автоматически подставятся № и дата договора с контрагентом, и будут выбраны основные реквизиты контрагента (если они были заданы в карте контрагента).
Для задания других (не основных) рекизитов контрагента воспользуйтесь кнопкой "..." возле поля "Банковский счет".

По умолчанию раскрыт полный список контрагентов. В этом списке можно найти необходимого Вам контрагента с помощью фильтров. Фильтры доступны по полям:
-Код контрагента
-Полное наименование
-ИНН
-КПП
-Юридический адрес.

Введите в одно или несколько из этих полей фрагмент данных нужного Вам контрагента и нажмите кнопку "Найти".
