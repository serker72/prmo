В этом диалоге Вы можете выбрать контрагента по приходу наличных.

Чтобы выбрать контрагента для прихода, выполните двойной клик по строчке с требуемым контрагентом.
При этом контрагент будет подставлен в приход.


По умолчанию раскрыт полный список контрагентов. В этом списке можно найти необходимого Вам контрагента с помощью фильтров. Фильтры доступны по полям:
-Код контрагента
-Полное наименование
-ИНН
-КПП
-Юридический адрес.

Введите в одно или несколько из этих полей фрагмент данных нужного Вам контрагента и нажмите кнопку "Найти".
