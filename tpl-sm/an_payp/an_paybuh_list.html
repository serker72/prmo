Что вы хотите распечатать?


Внимание! Вы выбираете даты передачи счетов в бухгалтерию. Дата установки галочки "счет в бухгалтерии" считается датой передачи счета в бухгалтерию.

%{else}% %{/if}%
Период







1. Фильтр "Период". По умолчанию показаны все %{if $is_incoming==1}%входящие%{else}%исходящие%{/if}% счета за все время работы программы.
При выборе фильтра "Только счета, переданные в бухгалтерию " становится доступным фильтр "Период." При выборе периода в режиме "Только счета, переданные в бухгалтерию " %{if $is_incoming==1}%входящие%{else}%исходящие%{/if}% счета отбираются по дате передачи в бухгалтерию, попадающую в выбранный период.

Если Вы не выбрали фильтр "Только счета, переданные в бухгалтерию ", то отбор счетов по периоду не проводится, и фильтр "Период" неактивен.

2. Фильтр "Контрагент". Вы можете осуществлять отбор %{if $is_incoming==1}%входящих%{else}%исходящих%{/if}% счетов по нескольким контрагентам.
Для этого в форму поиска нужно вписать фрагмент названия искомого контрагента. Программа выдаст список вариантов названий. Выберите в списке требуемого контрагента щелчком мыши по его названию. При необходимости отобрать в отчет счета по еще одному контрагенту - снова наберите фрагмент его названия и выберите контрагента из всплывшего списка вариантов названий контрагентов.
После выбора всех необходимых контрагентов нажмите кнопку .

3. Фильтр "Только счета, переданные в бухгалтерию " позволяет показать только те %{if $is_incoming==1}%входящие%{else}%исходящие%{/if}% счета, которые были переданы в бухгалтерию.
Фильтр "Только счета, не переданные в бухгалтерию" позволяет показать только те %{if $is_incoming==1}%входящие%{else}%исходящие%{/if}% счета, которые не были переданы в бухгалтерию.
Одновременно может быть отмечен только один из этих двух фильтров.

4. Фильтр "Оплаченные счета" позволяет показать только %{if $is_incoming==1}%входящие%{else}%исходящие%{/if}% счета со 100% оплатой.
Фильтр "Неоплаченные счета " позволяет показать только %{if $is_incoming==1}%входящие%{else}%исходящие%{/if}% счета, по которым не было оплат.
Одновременно может быть отмечен только один из этих двух фильтров.

5.Фильтр "Частично оплаченные счета " позволяет отобрать в отчет частично оплаченные счета.

Фильтры 3,4,5 могут сочетаться, например:

  • Сочетание фильтров "Только счета, переданные в бухгалтерию " и "Оплаченные счета" выдаст полностью оплаченные счета, переданные в бухгалтерию.
  • Сочетание фильтров "Только счета, переданные в бухгалтерию ", "Оплаченные счета" и "Частично оплаченные счета " выдаст полностью и частично оплаченные счета, переданные в бухгалтерию.
  • Сочетание фильтров "Только счета, переданные в бухгалтерию " и "Неоплаченные счета" выдаст переданные в бухгалтерию и неоплаченные счета.
