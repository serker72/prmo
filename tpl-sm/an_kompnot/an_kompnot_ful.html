Поиск по наименованию позиции:
Код позиции Наименование ед.изм. Товарная группа/ подгруппа 1 ур./ подгруппа 2 ур. ГОСТ/ТУ Длина, мм Ширина, мм Высота/ толщина, мм Диаметр, мм:



1. В отчет "Выполненные заявки" попадают заявки и их позиции, которые полностью завезены.
2. Фильтр "Период": заявки отбираются по заданным датам связанных поступлений, попадающим в выбранный период.
3. Фильтр "Покупатель". Вы можете осуществлять отбор по нескольким покупателям-контрагентам.
Для этого в форму поиска нужно вписать фрагмент названия искомого контрагента. Программа выдаст список вариантов названий. Выберите в списке требуемого контрагента щелчком мыши по его названию. При необходимости отобрать в отчет данные по еще одному контрагенту - снова наберите фрагмент его названия и выберите контрагента из всплывшего списка вариантов названий контрагентов.

4. Вы можете осуществлять отбор заявок по нескольким позициям. Для этого в форму поиска нужно вписать через точку с запятой фрагменты искомых названий и нажать кнопку .
5. Для того, чтобы осуществить расширенный поиск по позициям заявки (по коду позиции, ГОСТ/ТУ, длине, ширине, высоте, диаметру), необходимо нажать кнопку , заполнить соответствующие поля и нажать кнопку .
